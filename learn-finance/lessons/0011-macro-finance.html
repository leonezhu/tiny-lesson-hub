宏观金融：货币、央行与经济周期

TeachHub · 金融与投资 · 第11课 / 共12课 · 约19分钟阅读

本课目标：理解宏观金融的大图景——央行如何通过货币政策影响利率和资产价格，经济周期如何驱动金融市场，以及通货膨胀、汇率和国际资本流动如何塑造全球金融格局。学会用宏观视角理解「为什么有时所有资产一起涨跌」。

一、金融市场不是孤岛

在前十课中，我们学习了股票、债券、衍生品等「微观」金融工具。但任何资产的回报都不是孤立决定的——它们都嵌入在一个更大的宏观金融环境中：

  • 央行加息 → 所有资产价格受压（第1课的时间价值原理）
  • 经济衰退 → 企业盈利下降 → 股市下跌
  • 通货膨胀 → 债券实际收益缩水
  • 汇率波动 → 影响跨国投资回报
宏观金融 (Macro Finance)
研究宏观经济变量（利率、通胀、汇率、GDP增长）与金融市场之间相互关系的学科。它连接了宏观经济学（研究整体经济）和金融学（研究资产定价），关注货币政策、财政政策和国际资本流动如何影响资产价格和金融稳定。
如果说前10课教的是「如何评估一棵树」，宏观金融教的是「如何理解整片森林的气候」。再好的树木也逃不过旱灾。

二、央行与货币政策：金融市场的「总开关」

中央银行（如美联储、中国人民银行、欧洲央行）是现代金融体系的核心。它们的主要工具是货币政策——通过调控货币供给和利率来影响经济。

央行的双重目标（Dual Mandate）

  • 物价稳定：控制通货膨胀（大多数发达国家的目标通胀率约 2%）
  • 充分就业 / 经济增长：促进经济在潜在产出水平附近运行

这两个目标经常存在张力：刺激经济可能引发通胀，抑制通胀可能抑制增长。央行的艺术就在于平衡。

货币政策工具箱

1. 政策利率（最核心工具）
央行设定基准利率（如美联储的联邦基金利率），影响整个经济体的借贷成本。
降息 → 借贷更便宜 → 消费和投资增加 → 经济扩张，但可能推高通胀
加息 → 借贷更贵 → 经济降温，通胀受控，但可能引发衰退
2. 公开市场操作（Open Market Operations）
央行在市场上买卖国债，调节银行体系的流动性。
买入国债 → 释放货币 → 利率下降
卖出国债 → 回收货币 → 利率上升
3. 量化宽松 / 量化紧缩（QE / QT）
当利率已经降到零附近、传统工具失效时，央行直接大规模购买长期债券（QE），压低长期利率。
2008 年金融危机和 2020 年新冠疫情期间，美联储进行了史无前例的 QE，资产负债表从不到 1 万亿膨胀到近 9 万亿美元。2022 年开始 QT（缩表）。
4. 存款准备金率
要求商业银行必须按存款比例留存准备金。降低准备金率 → 银行可贷资金增加 → 货币乘数扩大。这是中国人民银行常用的工具。

三、利率如何传导到金融市场？

央行的每一次加息或降息，都会通过多条渠道传导到所有金融市场：

1. 贴现率渠道（估值渠道）

这是最直接的传导机制（也是第1课的核心内容）：加息 → 贴现率上升 → 所有未来现金流的现值下降 → 股票、债券、房产价格下跌。成长股（未来现金流集中在中长期）对加息尤其敏感。

2. 信贷渠道

加息 → 银行贷款利率上升 → 企业融资成本上升 → 投资减少、利润下降 → 股价下跌。同时，浮动利率债务的偿付压力增大，违约风险上升。

3. 汇率渠道

加息 → 本国利率高于他国 → 资本流入 → 本币升值。本币升值利好进口商、利空出口商。

4. 资产价格渠道（财富效应）

加息 → 资产价格下跌 → 居民财富缩水 → 消费意愿下降 → 经济进一步降温。这形成了一个正反馈循环。

2022 年案例：美联储为对抗通胀，全年加息 425 个基点（从 0.25% 到 4.5%）。结果：标普500指数下跌约 19%，美国综合债券指数下跌约 13%（百年最差债券年），比特币暴跌 65%。加息几乎打击了所有资产类别——这就是「宏观压制」。

四、经济周期与金融市场的舞蹈

经济不是线性增长的，而是呈周期性波动。一个典型的经济周期包括四个阶段，每个阶段对不同资产的影响不同：

阶段一：复苏期（Recovery）
刚走出衰退，经济开始加速。央行维持低利率。
最佳资产：股票（尤其周期股、小盘股）、高收益债券
逻辑：低利率 + 盈利改善 = 股市黄金期
阶段二：扩张期（Expansion / Overheating）
经济强劲增长，但通胀开始升温。央行开始加息。
最佳资产：股票（向防御股轮动）、大宗商品
风险：估值偏高，加息开始压制市场
阶段三：滞胀期（Stagflation）
通胀高企但增长放缓——最痛苦的组合。
最佳资产：现金、抗通胀资产（黄金、TIPS）、短久期债券
风险：股债双杀，1970 年代美国经历了长期滞胀
阶段四：衰退期（Recession）
经济负增长，企业盈利暴跌，失业率上升。央行大幅降息救市。
最佳资产：长期国债（降息直接推高债券价格）、高质量防御股、现金
逻辑：降息利好债券；资金避险涌入安全资产
美林时钟（Merrill Lynch Investment Clock）将上述逻辑总结为一个框架：根据经济增长和通胀的不同组合，在债券、股票、大宗商品和现金之间轮动配置。这是宏观资产配置的经典工具。

五、通货膨胀：金融资产的隐形杀手

通胀是宏观金融最重要的变量之一。它对金融的影响是全方位的：

通胀的两种类型

  • 需求拉动型：经济过热，需求超过供给 → 价格普涨。对策：加息降温
  • 成本推动型：供给侧冲击（如石油危机、供应链断裂）→ 生产成本上升 → 价格上涨。这种通胀更难治理——加息抑制需求但不解决供给问题

通胀对不同资产的影响

资产类别 通胀影响
长期固定利率债券最差（实际收益被侵蚀）
现金差（购买力缩水）
成长股差（加息压估值）
价值股 / 资源股较好
不动产（房地产）较好（租金和房价随通胀涨）
黄金 / 大宗商品最好（传统抗通胀工具）
通胀保值债券（TIPS）好（本金随通胀调整）
1970年代大通胀：美国 CPI 一度超过 14%。十年间标准普尔500名义回报为正，但扣除通胀后实际回报接近零。同期黄金上涨超过 20 倍。这就是为什么「通胀是债券的天然敌人，黄金的天然朋友」。

六、汇率与国际资本流动

在全球化的金融市场中，汇率是连接各国经济的纽带。汇率变动直接影响跨国投资的回报：

跨国投资总回报 = 资产本币回报 ± 汇率变动

如果你投资美国标普500指数赚了 10%，但美元对人民币贬值了 8%，你的实际人民币回报只有约 2%。这就是汇率风险。

影响汇率的主要因素

  • 利率差：高利率国家吸引资本流入 → 本币升值（「利差交易」Carry Trade）
  • 通胀差：高通胀国家货币购买力下降 → 本币贬值（购买力平价理论）
  • 贸易差额：贸易顺差国货币趋于升值
  • 经济增长：强劲增长吸引外资 → 本币升值
  • 政治和制度稳定性：稳定的政治环境吸引「避险资本」
不可能三角 (Impossible Trinity)
一个国家不可能同时拥有以下三者：① 固定汇率、② 资本自由流动、③ 独立的货币政策。最多只能选两个。例如：香港选择了固定汇率 + 资本自由流动 → 放弃独立货币政策（利率跟随美国）；中国选择了独立货币政策 + 资本管制 → 牺牲汇率弹性（部分管控）。

七、系统性风险与金融危机

宏观金融最关注的风险是系统性风险（Systemic Risk）——单个机构的困境传导到整个金融体系，导致系统性崩溃的风险。

2008 年全球金融危机的宏观金融教训

  • 杠杆过度：次贷危机前，金融机构杠杆率极高，微小损失被放大
  • 传染效应：雷曼倒闭 → 信任崩溃 → 银行间市场冻结 → 全球信贷瘫痪
  • 影子银行：不受监管的金融中介放大了系统性风险
  • 政策应对：央行作为「最后贷款人」提供紧急流动性、量化宽松，阻止了全面崩溃
金融危机的本质是信心危机。当信任消失时，即使健康的机构也会因流动性枯竭而倒下。这就是为什么央行和政府必须在危机中果断干预——恢复信心比拯救个别机构更重要。

跨学科链接

  • 宏观经济学：IS-LM 模型、总供给-总需求模型、菲利普斯曲线（通胀-失业权衡）
  • 国际经济学：购买力平价、蒙代尔-弗莱明模型、国际收支
  • 政治学：中央银行独立性、财政-货币政策协调
  • 历史学：大萧条、大通胀、亚洲金融危机、2008 年金融危机——历史是最好的宏观金融教科书
本课核心收获：
  1. 宏观金融连接宏观经济与金融市场——利率、通胀、汇率塑造所有资产的回报
  2. 央行是金融市场的「总开关」：通过利率、QE/QT、准备金率等工具影响经济
  3. 加息通过贴现率、信贷、汇率和财富效应四条渠道传导到所有资产
  4. 经济周期分为复苏、扩张、滞胀、衰退——每个阶段最佳资产配置不同（美林时钟）
  5. 通胀是债券和现金的隐形杀手，是黄金和大宗商品的朋友
  6. 汇率风险是跨国投资不可忽视的成本；不可能三角约束国家政策选择
  7. 金融危机的本质是信心危机——系统性风险传染速度远超想象
