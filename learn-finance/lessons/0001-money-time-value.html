货币的时间价值

TeachHub · 金融与投资 · 第1课 / 共12课 · 约15分钟阅读

本课目标：掌握金融学最核心的概念——「今天的一块钱比明天的一块钱更值钱」。理解现值（PV）、终值（FV）、复利和贴现的数学原理，学会用时间价值思维评估一切投资决策。

一、金融学的第一原理

如果有人问你：「现在给你 10000 元，还是一年后给你 10000 元？」你会选哪个？

任何理性人都会选现在。为什么？因为：

  • 你现在拿到 10000 元，可以存入银行，一年后变成 10300 元（假设利率 3%）
  • 通货膨胀会让一年后的 10000 元购买力下降
  • 未来有不确定性——对方可能违约、你可能忘记、世界可能变化

这就是金融学的第一原理：货币具有时间价值。时间本身就是金钱。

「今天的一块钱比明天的一块钱更值钱。」这是所有金融估值、投资决策、贷款定价的基石。
货币的时间价值 (Time Value of Money, TVM)
货币在不同时点上的价值是不同的。由于资金具有赚取收益的潜力（机会成本），同样数额的货币，现在比未来更有价值。

二、终值与复利：让钱生钱的力量

如果你今天投资 PV（现值），年利率为 r，投资 n 年，未来你能拿到多少钱？这就是终值（Future Value, FV）。

FV = PV × (1 + r)n

这个公式背后是复利（Compound Interest）的力量。复利意味着你的利息也会产生利息。爱因斯坦据说称复利为「世界第八大奇迹」。

单利 vs 复利

  • 单利：只对本金计算利息。FV = PV × (1 + r × n)
  • 复利：对本金和累积利息一起计算利息。FV = PV × (1 + r)n

差距有多大？假设你投资 10000 元，年利率 8%，30 年：

  • 单利：10000 × (1 + 0.08 × 30) = 34000 元
  • 复利：10000 × (1.08)30 = 100627 元

复利的结果是单利的近 3 倍。这就是为什么巴菲特说：「我的财富来自美国、长寿和复利。」

72 法则：用 72 除以年利率，约等于资金翻倍所需的年数。利率 8% → 72÷8 = 9 年翻倍。利率 12% → 6 年翻倍。利率 6% → 12 年翻倍。这是一个快速估算复利力量的心算工具。

三、现值与贴现：未来的钱值多少？

终值问的是「现在的钱未来值多少」，现值问的反过来：「未来的钱现在值多少？」

这就是贴现（Discounting）——把未来的现金流「折回」到现在。贴现是金融估值的逆向工程。

PV = FV / (1 + r)n

这里的 r 叫做贴现率（Discount Rate），它反映了：

  • 资金的机会成本（你本可以把钱投到别处）
  • 通货膨胀的侵蚀
  • 风险溢价（未来现金流越不确定，贴现率越高）
贴现率 (Discount Rate)
将未来现金流折算为现值时所使用的利率。贴现率越高，未来现金流的现值越低。它综合反映了时间偏好、机会成本和风险补偿。

一个关键直觉：贴现率越高，现值越低

假设你一年后将收到 10000 元：

  • 贴现率 3% → PV = 10000 / 1.03 = 9709 元
  • 贴现率 8% → PV = 10000 / 1.08 = 9259 元
  • 贴现率 15% → PV = 10000 / 1.15 = 8696 元

贴现率从 3% 涨到 15%，同一笔未来款项的现值缩水了 10% 以上。这就是为什么加息对资产价格是毁灭性的——后面课程会反复看到这一点。

四、年金与永续年金

现实中，你很少只收到一笔钱。更常见的是一系列定期等额的现金流——这叫年金（Annuity）。

年金 (Annuity)
一系列在固定间隔期内、等额收付的现金流。例如每月房贷还款、每年退休金、每期彩票奖金等。

年金的现值公式：

PV年金 = C × [1 - (1+r)-n] / r

其中 C 是每期支付额，r 是每期利率，n 是期数。

如果年金永远持续下去（n → ∞），就变成了永续年金（Perpetuity）：

PV永续 = C / r

这个优雅的公式告诉你：一只每年支付 5 元股息、永远增长的股票，在 10% 的要求回报率下，值 5/0.10 = 50 元。这就是股票估值的雏形（第7课详述）。

现实应用——英国统一公债：英国政府曾发行过一种叫 Consol 的永续年金债券，永不还本，只付固定利息。这是金融史上最经典的永续年金案例，直到 2015 年才被全部赎回。

五、现实案例：为什么贷款买房总感觉「亏了」？

假设你贷款 100 万元买房，30 年期，年利率 5%，等额本息还款。每月还约 5368 元，30 年总共还款 193 万元——你支付了 93 万利息！

很多人看到这个数字会愤怒：「我借 100 万，结果还了近 200 万？银行太黑了！」但用时间价值思维看，这完全合理：

  • 你借到的 100 万是今天的现值
  • 你还的 193 万是分散在 30 年里的终值，其中每一年的钱都要贴现回今天才公平
  • 如果你把 30 年的所有月供按 5% 贴现回今天，总和正好等于 100 万
贷款的「利息总额」看起来吓人，但那是名义总和。用时间价值贴现后，你付出的实际价值等于你借到的本金。利息是你使用资金时间的合理代价。

本课核心收获：
  1. 货币有时间价值——今天的一块钱比明天的一块钱更值钱
  2. 复利是「世界第八大奇迹」——时间越长，威力越大（72法则）
  3. 贴现是估值的逆向工程——未来的钱必须折算回现在才有意义
  4. 贴现率越高，未来现金流的现值越低——这是加息打击资产价格的根源
  5. 年金和永续年金公式是房贷、退休金、股票估值的基础工具
