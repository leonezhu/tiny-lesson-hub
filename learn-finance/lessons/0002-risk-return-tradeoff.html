第2课 — 风险与收益的权衡

金融学 · 第2课 / 共12课
本课目标：理解风险与收益的关系，学会用夏普比率评估投资回报，明白为什么高收益必然伴随高风险——以及你该如何应对。

一、核心法则：没有免费的午餐

金融学的第一个真理很简单：高收益来自高风险。这不是警告，而是描述。市场不会因为你想要更多钱就给你更多回报。

想象两个投资机会：

选项 A：国债

中国政府发行10年期国债，年化收益率约 2.5%。几乎无违约风险。

选项 B：初创公司股权

你投资一家AI创业公司，可能翻倍，也可能归零。预期年化 15%–30%，但波动极大。

为什么选项 B 的"预期"收益更高？因为你在承担更多的不确定性。风险溢价（Risk Premium）就是市场为补偿你承担的额外风险而支付的"加班费"。

预期收益率 = 无风险利率 + 风险溢价
例：国债 2.5% + 股票风险溢价 5% ≈ 预期股市回报 7.5%

二、风险的量化：标准差与波动率

风险不是感觉，是可以测量的。最常用的指标是标准差（Standard Deviation）——它衡量收益偏离平均值的程度。

两只基金年均回报都是 10%：
• 基金A的标准差是 3%（每年回报在 7%–13% 之间波动）
• 基金B的标准差是 25%（每年可能在 -15% 到 +35% 之间跳跃）

平均回报一样，但风险天差地别。理性投资者会选择A。

这里的关键洞察是：你关心的不是总波动，而是下行风险。涨过头了你不会难受，跌过头了你睡不着觉。这就是为什么专业投资者用下行标准差而不是普通标准差来衡量风险。

三、夏普比率：性价比之王

如果两只基金回报不同、风险也不同，怎么比？答案是夏普比率（Sharpe Ratio）——它回答了一个简单的问题：每承担一单位风险，我多赚了多少钱？

夏普比率 = (投资组合收益率 − 无风险利率) / 组合标准差

例：基金A 回报10%，标准差3%，无风险利率2.5%
夏普比率 = (10% − 2.5%) / 3% = 2.50

基金B 回报15%，标准差25%，无风险利率2.5%
夏普比率 = (15% − 2.5%) / 25% = 0.50

尽管B回报更高，但A的"风险调整后收益"远超B。

夏普比率大于1被认为是好的，大于2是优秀的。这个指标把"赚了多少"和"担了多少心"合二为一，是基金经理最害怕被问到的数字之一。

四、分散化：唯一的免费午餐

Harry Markowitz 因此获得诺贝尔奖，因为他发现了一个反直觉的事实：分散化可以降低风险而不降低预期收益。

关键概念是相关性（Correlation）：

  • 相关系数 +1：两只资产同涨同跌（完全正相关）
  • 相关系数 0：彼此无关（不相关）
  • 相关系数 -1：一只涨另一只跌（完全负相关）
如果你只持有科技股，市场下跌时你全部亏损。
如果你同时持有科技股（高增长、高波动）、国债（低收益、低风险）、黄金（抗通胀），那么当科技股下跌时，资金可能流向国债和黄金，你的整体波动就会大幅降低。

这就是分散化的魔力：用不相关的资产组合，降低整体风险。

但要注意：分散化不能消除系统性风险（Systematic Risk）——整个市场一起崩盘的时候，什么资产都救不了你。它能消除的是非系统性风险（Idiosyncratic Risk）——单个公司或行业特有的风险。

五、风险偏好：认识你自己

同样的风险调整回报，不同的人感受完全不同。这取决于你的风险容忍度：

投资者类型典型配置最大回撤容忍
保守型80% 债券 / 20% 股票-5%
稳健型60% 股票 / 40% 债券-15%
激进型90% 股票 / 10% 另类-30%

选择错误风险偏好的后果很严重：一个保守的人在2008年持有100%股票组合，可能在最低点恐慌卖出，锁定永久性损失。这比资产本身跌多少更致命——行为偏差往往比市场风险造成更大的伤害。
