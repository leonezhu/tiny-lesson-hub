行为金融学：人为何不理性

TeachHub · 金融与投资 · 第10课 / 共12课 · 约17分钟阅读

本课目标：理解传统金融学的「理性人」假设为何在现实中反复失效。学习认知偏差如何影响投资决策，认识市场泡沫与非理性的成因，掌握对抗自身心理陷阱的实用策略。

一、传统金融学的致命假设

到目前为止，我们学到的金融理论——CAPM、有效市场假说、期权定价模型——都建立在一个核心假设上：投资者是完全理性的。理性人会：

  • 总是追求效用最大化
  • 能够无偏地处理所有信息
  • 对风险有一致的态度
  • 不会被情绪干扰

但现实是：人是人，不是电子表格。

行为金融学 (Behavioral Finance)
将心理学（尤其是认知心理学和行为经济学）与金融学结合的交叉学科。研究认知偏差、情绪和社会因素如何影响投资者决策和资产价格。代表人物包括丹尼尔·卡尼曼（2002年诺贝尔经济学奖）、理查德·塞勒（2017年诺贝尔经济学奖）和罗伯特·席勒（2013年诺贝尔经济学奖）。

行为金融学并非要推翻传统金融学，而是补充它。理性模型告诉我们「应该怎么做」，行为金融学告诉我们「实际上人们是怎么做的」以及「为什么会出错」。

市场在大部分时间里接近有效，但在极端情况下——泡沫、崩盘、羊群效应——人类的系统性非理性会主导价格。

二、前景理论：损失比获得更痛

传统金融学用期望效用理论描述人的决策。1979 年，卡尼曼和特沃斯基提出了前景理论（Prospect Theory），彻底改变了我们对风险决策的理解。其核心发现：

同样金额的损失带来的痛苦 ≈ 2~2.5 倍于同等金额的收益带来的快乐

这意味着：人类天生是「损失厌恶」的。这种不对称有深远的金融含义。

损失厌恶导致的非理性行为

你持有两只股票：A 赚了 50%，B 亏了 50%。你需要卖掉一只。大多数人会怎么选？

  • 理性答案：卖掉未来前景更差的那只（与盈亏无关）
  • 实际行为：大多数人卖掉 A（「锁定收益」），留着 B（「等它涨回来」）

这就是著名的处置效应（Disposition Effect）——投资者倾向于「过早卖出盈利股、过久持有亏损股」。根源就是损失厌恶：承认亏损的痛苦太大了，所以一直等「回本」。

赌场心理学：赌徒在输钱后往往「加倍下注」想回本——这是损失厌恶的极端体现。在投资中，补仓「抄底」亏损股而不愿止损，本质上也是同一种心理。

三、六大认知偏差：投资中的隐形陷阱

1. 锚定效应 (Anchoring)
人在做判断时，会过度依赖最先获得的信息（「锚」），即使这个信息无关紧要。

投资案例：你以 100 元买入股票，跌到 60 元。你判断「便宜了」（以 100 元为锚）。但如果股票本身只值 40 元呢？你的买入价不应影响它的内在价值——但你就是忍不住以此为参照。
2. 确认偏差 (Confirmation Bias)
人们倾向于寻找、解读和记住那些支持自己已有观点的信息，忽略矛盾的证据。

投资案例：你看好某只股票后，只看利好新闻，对利空消息充耳不闻。这导致「信息茧房」，越看越觉得自己对——直到崩盘。
3. 过度自信 (Overconfidence)
绝大多数人（研究表明超过 80%）认为自己的驾驶能力「高于平均水平」——这在数学上不可能。在投资中，人们系统性地高估自己选股和择时的能力。

投资案例：频繁交易的散户平均收益率远低于买入持有者。每次交易都源于「我觉得我能赚」，但交易成本和择时失误反而侵蚀了收益。男性比女性更容易过度自信，因此男性散户平均表现更差。
4. 从众效应 / 羊群效应 (Herding)
人有强烈的群体归属需求。当不确定时，我们会观察别人的行为来决定自己怎么做。

投资案例：比特币、GameStop、NFT——当「所有人都在赚钱」时，FOMO（错失恐惧症）驱使人们冲入市场。但羊群效应往往在顶部最强：当最后一批人入场时，聪明钱已经在离场了。
5. 近因偏差 (Recency Bias)
人们过度看重最近发生的事，忽略长期的历史规律。

投资案例：牛市持续了三年，人们就觉得「股市永远涨」——于是在顶部满仓。反之，熊市中人们觉得「股市永远跌」——于是在底部清仓。这正是「高买低卖」悲剧的心理根源。
6. 心理账户 (Mental Accounting)
人在心里把钱分成不同的「账户」，对不同来源的钱有不同的花费倾向——尽管钱是同质的。

投资案例：年终奖「轻松赚到」的钱，人们倾向于冒险花掉或投机；而辛苦工资则谨慎储蓄。理性行为应该是：所有钱的边际效用是一样的，与来源无关。

四、市场层面的非理性：泡沫与异象

泡沫的形成机制

个别投资者的非理性叠加起来，就会形成市场泡沫。席勒在《非理性繁荣》中描述了泡沫的典型循环：

  1. 结构性变化：新技术、新政策或新市场出现（如互联网、房地产）
  2. 早期成功：先行者确实赚到了钱
  3. 媒体放大：成功故事被广泛传播
  4. 羊群涌入：FOMO 驱动更多人入场
  5. 价格上涨自我强化：涨 → 吸引 → 更多买入 → 继续涨
  6. 理性参与者离场：聪明钱开始套现
  7. 崩盘：某个事件触发恐慌抛售，踩踏开始
历史泡沫案例：1637 年荷兰郁金香泡沫、1720 年英国南海泡沫、2000 年互联网泡沫、2008 年房地产泡沫、2021 年加密货币泡沫。模式惊人地一致——唯一的区别是底层资产的「故事」不同。

市场异象（Anomalies）

有效市场假说预测市场不应存在系统性可预测的模式，但实证研究发现了一些持续存在的「异象」：

  • 动量效应：过去 6-12 个月表现好的股票，未来短期内倾向于继续好
  • 价值效应：低估值（低 P/B、低 P/E）股票长期表现优于高估值股票
  • 小市值效应：小公司股票长期平均回报高于大公司
  • 日历效应：「一月效应」「星期五效应」等季节性模式

这些异象部分可以用行为偏差来解释：例如动量效应源于反应不足（投资者对好消息消化太慢），价值效应部分源于过度反应（投资者对坏消息反应过度，导致价值股被低估）。

五、如何对抗自己的非理性？

行为金融学不只是「诊断」，更提供了处方。以下是基于行为金融学研究的实用建议：

1. 系统化投资规则

用规则取代情绪。例如：每月固定金额定投指数基金（定投策略），不论市场涨跌。这能有效对抗择时的诱惑和近因偏差。

2. 预先承诺（Pre-commitment）

在冷静时做好决策：写下投资计划、设定自动再平衡规则。当市场恐慌或狂热时，你已经有了「自动驾驶」方案。

3. 写投资日记

记录每次买卖的理由。回顾时你会发现自己的判断模式——尤其是反复犯的错误。这是对抗确认偏差和后见之明偏差的有效工具。

4. 逆向思考

问自己：「如果我要推翻自己的观点，最有力的论据是什么？」主动寻找反面证据，对抗确认偏差。

5. 关注成本而非收益

你无法控制市场收益，但可以控制交易频率、费用和税收。减少交易就是减少过度自信犯错的机会。

认识偏差不能消除偏差。最有效的策略是设计系统来绕开偏差，而非依赖意志力在每次决策中对抗人性。

跨学科链接

  • 认知心理学：双系统理论（系统1快思考、系统2慢思考）——卡尼曼《思考，快与慢》
  • 进化生物学：损失厌恶可能是进化的产物——在远古环境中，失去食物的后果比获得额外食物的好处更致命
  • 神经科学：fMRI 研究显示，面对亏损时大脑杏仁核（恐惧中心）强烈激活
  • 行为经济学：助推（Nudge）理论——如何用「选择架构」引导更好的决策
本课核心收获：
  1. 传统金融学的「理性人」假设在现实中经常失效——人是有限理性的
  2. 前景理论揭示：损失的痛苦约为同等收益快乐的 2-2.5 倍（损失厌恶）
  3. 处置效应：投资者倾向过早卖出盈利、过久持有亏损
  4. 六大认知偏差：锚定、确认、过度自信、从众、近因、心理账户
  5. 泡沫遵循可预测的心理循环：新故事 → 早期赚钱 → 媒体放大 → 羊群涌入 → 崩盘
  6. 对抗偏差最有效的方法不是意志力，而是设计系统化投资规则
