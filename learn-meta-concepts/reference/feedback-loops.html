反馈回路

元概念速查卡 · 课程 4 · ~3 minutes

核心定义

系统的输出作为未来输入返回到系统中，形成循环因果链。正反馈放大偏差，负反馈抵消偏差。

核心特征时间延迟

正反馈

结构：A → B → C → 放大 A

  • 复利：A = P(1+r)^n
  • 流行病传播：R0 值决定指数增长速度
  • 社交网络效应：用户越多→价值越大→更多用户
  • 2008 金融危机：房价下跌→抵押贬值→抛售→再下跌

负反馈

结构：A → B → C → 抑制 A

  • 体温调节：37°C 目标值，误差 <0.1°C
  • 血糖调节：进食→血糖↑→胰岛素→吸收→血糖↓
  • 恒温器：设定目标值→检测偏差→纠正
  • 捕食者-猎物：Lotka-Volterra 周期振荡

延迟 × 反馈 = 系统行为

  • 无延迟：稳定
  • 适度延迟：振荡
  • 过长延迟：混沌 / 矫枉过正

Sterman：管理失败的根源 = 对反馈延迟的误判

关键人物

  • Wiener (1948) — 控制论创始人
  • Meadows — 《系统之美》《增长极限》
  • Sterman — 《Business Dynamics》，反馈延迟量化
  • Lovelock — 盖亚假说
  • Soros — 反身性理论

常见陷阱

  • 线性思维（A→B）忽略回路结构
  • 症状解：处理表面症状却忽略深层反馈
  • Goodhart 定律：指标成为目标后失效
  • 正反馈失控：没有负反馈制衡的增长
