熵（Entropy）— 快速参考卡

元概念 · 课程 2 · 参考速查

一句话定义

熵是描述系统无序程度或不确定性的量化指标。孤立系统总是自发趋向高熵（更混乱）状态。

三个公式

S = kB · ln(Ω)

玻尔兹曼熵：Ω = 微观状态数，kB = 1.38×10⁻²³ J/K

H = −Σ p(x) · log₂ p(x)

香农信息熵：p(x) = 事件 x 的概率，H = 比特

ΔS ≥ ∫ dQ/T

克劳修斯不等式：dQ = 热交换，T = 绝对温度

三种视角

  • 热力学熵：能量分散程度。热量从高温→低温，不可逆。
  • 统计熵：微观排列数量。有序态少，无序态多，系统自然走向多的那一边。
  • 信息熵：预测难度。越不可预测 → 信息熵越高。

关键规则

  • 孤立系统：熵永不减少（热力学第二定律）
  • 局部熵减是允许的，但总熵一定增加
  • 有序结构的维持需要持续的能量输入
  • 信息熵定义了无损压缩的理论下限

跨领域应用

  • 物理：热力学、统计力学、宇宙学（热寂）
  • 信息：数据压缩、密码学、通信理论
  • 生物：生命以负熵为食（薛定谔）
  • 管理：组织熵增 → 流程退化、文档过时
  • 经济：市场均衡 = 信息熵最大化

常见误解

  • ❌ "熵增 = 一切变坏" → ✅ 熵是物理描述，不是价值判断
  • ❌ "生命违反熵增" → ✅ 生命是局部低熵，整体熵仍在增加
  • ❌ "熵就是混乱" → ✅ 更准确地说，熵是"微观状态的多寡"
