三次全球化大周期

第一波 1870-1914（金本位时代）

  • 贸易量增长10倍，5500万欧洲人移民
  • 技术驱动：蒸汽船、铁路、电报
  • 终结：一战 → 边境关闭 → 贸易萎缩

第二波 1945-2008（布雷顿森林体系）

  • IMF + 世界银行 + GATT/WTO
  • "嵌入式自由主义"：自由贸易 + 福利国家缓冲
  • 2001中国加入WTO → 15亿人进入全球劳动力市场
  • 1990-2008超级全球化巅峰

第三波 2008至今（再全球化）

  • 2008金融危机 → 贸易保护主义抬头
  • 贸易/GDP比重下降后回升至历史高位
  • 供应链从"效率优先"转向"安全+效率"
  • 格局：单一全球市场 → 多个区域网络

两个核心分析框架

罗德里克不可能三角：民主政治、国家主权、超全球化——三选二。
民主 + 主权 → 牺牲超级全球化 = 当前脱钩趋势的必然性
波兰尼双向运动：市场过度扩张 → 社会保护性反弹（系统自我调节，非道德判断）。
19世纪自由贸易 → 20世纪初社会立法 → 1990年代超级全球化 → 2010年代民粹反弹

关键数据点

贸易与关税

  • 1870-1913：全球贸易量增长10倍
  • 1930斯姆特-霍利关税法 → 全球贸易萎缩66%
  • GATT谈判：工业品关税从40%降至不到5%
  • WTO成员164个，覆盖全球98%贸易
  • 2024美国对华关税覆盖率约20%（贸易战前3%）

分配效应（大象曲线）

  • 最大受益者：亚洲新兴中产阶级 + 全球前1%
  • 发达国家中产收入增长停滞
  • 美国1979-2009失去约600万制造业岗位
  • 英国2016脱欧、美国2016特朗普当选

脱钩四维度

  • 技术脱钩：半导体出口管制、华为禁令
  • 供应链："中国+1"、近岸外包到墨西哥/越南
  • 金融：去风险（de-risking）、中概股审计
  • 数据：TikTok之争、跨境数据流动限制
