Reference Card · 工业革命

五大先决条件（为什么是英国？）

  • 丰富的浅层煤炭资源
  • 农业革命释放农村劳动力
  • 产权保护 + 股票市场（1688光荣革命后）
  • 殖民帝国的原材料与市场网络
  • 启蒙文化与工匠交流机制

第一次工业革命（1760-1840）

核心：纺织机械 + 蒸汽机

  • 珍妮纺纱机（1764）— 8倍效率
  • 水力纺纱机（1769）— 工厂制度诞生
  • 瓦特改良蒸汽机（1769）— 通用动力
  • 动力织布机（1785）— 机械化织布

第二次工业革命（1870-1914）

核心：科学引领产业

  • 贝塞麦炼钢（1856）— 钢铁大众化
  • 电力普及 — 动力分散化
  • 化学合成 — 全新产业门类
  • 内燃机与石油 — 汽车与航空
  • 电报与电话 — 即时通信

关键数字

  • 全球城市化率：3%（1800）→ 14%（1900）
  • 曼彻斯特人口：1万（1772）→ 35万（1851）
  • CO₂浓度：280ppm（工业革命前）→ 420ppm+（今天）
  • 瓦特蒸汽机效率提升：4倍以上

社会结构剧变

  • 资产阶级崛起 — 资本取代土地成为权力来源
  • 无产阶级形成 — 出卖劳动力的新型阶层
  • 家庭与工作分离 — 新的社会时空秩序
  • 女性角色 — 中产阶级"男主外" vs 工人阶级女工/童工

大分流主要解释

  • 地理论（戴蒙德）— 欧亚大陆轴向
  • 制度论（诺斯）— 产权与创新激励
  • 能源论（诺德豪斯）— 煤炭可及性
  • 文化论（韦伯）— 新教伦理
