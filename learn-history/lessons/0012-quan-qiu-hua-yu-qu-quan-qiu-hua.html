全球化与去全球化

第12课 · ~12分钟 · 经济史与地缘政治

学完本课，你将理解全球化不是一个单向进程，而是一个在扩张与收缩之间摇摆的周期；能运用"罗德里克不可能三角"和"波兰尼双向运动"框架分析当下的中美脱钩、英国脱欧等现象；并理解"去全球化"不是全球化的终结，而是它的重组。

为什么全球化会"逆涨"？

1990年代到2008年，全球化进入了"超级全球化"阶段：全球贸易占GDP比重超过60%，跨国供应链深度交织，资本和信息自由流动。但2008年金融危机之后，钟摆开始回摆——英国脱欧、美国加征关税、供应链"近岸化"，"去全球化"成为新闻常客。

但历史告诉我们：全球化不会消失，它只会改变形态。理解这个周期的规律，比判断"全球化结束了吗"这个问题重要得多。

全球化三次大周期

第一波：1870–1914

技术驱动：蒸汽船、铁路和电报将世界的地理距离压缩到前所未有的程度。

  • 全球贸易量增长约10倍
  • 约5500万欧洲人移民到美洲和澳大利亚
  • 利物浦与芝加哥的小麦价差从57.6%降至15.6%
  • 金本位制度提供了稳定的国际货币锚

终结原因：第一次世界大战（1914）以暴力方式终结了这波全球化。各国关闭边境、实行贸易管制、放弃金本位。1930年美国斯姆特-霍利关税法引发全球报复性加税，全球贸易在1929-1933年间萎缩约66%。

第二波：1945–2008

制度框架：二战后美国主导构建了布雷顿森林体系——IMF维持汇率稳定，世界银行资助重建，GATT（后升级为WTO）逐步降低关税。这一模式被称为"嵌入式自由主义"：自由贸易被嵌入在各国福利国家的缓冲机制中。

  • GATT八轮谈判将工业品平均关税从40%降至不到5%
  • 1995年WTO成立，引入争端解决机制
  • 2001年中国加入WTO——约15亿人进入全球劳动力市场
  • 1990-2008年金融全球化达到巅峰

积累的矛盾：发达国家制造业岗位持续流失（美国1979-2009年失去约600万个），资本回报率远超劳动回报率，收益集中在沿海都市和资本所有者手中。

第三波：2008至今——"再全球化"

转折点：2008年金融危机暴露了金融全球化的系统性风险。雷曼兄弟倒闭在几周内传导到全球每个角落。

  • 全球贸易占GDP比重在2008年后下降，但2022年已回到历史高位附近
  • 供应链从"效率优先"转向"安全+效率"双目标
  • 服务贸易（数字服务、金融、教育）快速增长
  • 格局变化：从"单一全球市场"转向"多个区域网络"

核心框架：为什么周期会反复

罗德里克"不可能三角"

经济学家丹尼·罗德里克提出了理解全球化的最深刻框架：民主政治、国家主权和超全球化——三者最多只能同时存在两个。

  • 超级全球化 + 民主 → 必须牺牲部分国家主权（如欧盟）
  • 超级全球化 + 国家主权 → 必须牺牲民主（如威权国家的出口导向模式）
  • 民主 + 国家主权 → 必须牺牲超级全球化（如当前的脱钩趋势）

当全球化进程太快、分配太不均时，民主机制会通过选票将其纠正——英国脱欧和特朗普当选正是这种纠正的体现。

波兰尼的"双向运动"

卡尔·波兰尼在《大转型》（1944）中提出：每当市场力量过度扩张、侵入社会生活每一个角落时，社会就会产生保护性反弹。这不是"进步"或"落后"的问题，而是系统自我调节的必然。

  • 19世纪自由贸易 → 20世纪初社会立法和工会运动
  • 1990年代超级全球化 → 2010年代民粹反弹
  • 每一次反弹都可能导致全球化进程的调整，而非终结
核心洞察：全球化不是线性进步，而是钟摆。每一次"逆全球化"浪潮都有相似的触发机制：经济不平等加剧、本地社群被全球化冲击、政治精英对民意的忽视。理解这个周期，是理解当下中美脱钩、英国脱欧、右翼民粹崛起的历史钥匙。

中美脱钩：21世纪的去全球化

2018年开始的中美贸易战标志着系统性脱钩的开始。到2024年，美国对华关税覆盖率已从贸易战前的约3%升至约20%。但脱钩远超关税范畴：

  • 技术脱钩：美国对中国半导体出口管制，限制华为获取先进芯片和EDA工具
  • 供应链重组："中国+1"战略、近岸外包到墨西哥和越南
  • 金融脱钩：中概股审计纠纷、去风险（de-risking）叙事
  • 数据脱钩：TikTok的命运之争、跨境数据流动限制
关键区别：不同于1914年的暴力断裂和1930年代的全面闭关锁国，当前的"去全球化"更像是一种结构性的缓慢分离。全球经济仍深度互联，但在关键技术领域（半导体、AI、量子计算）正在形成多个竞争性技术阵营。

全球化与网络效应

从网络效应角度看，全球供应链形成了一个高度互联的网络。节点的故障（如疫情、地震、政治冲突）可以通过网络传播为系统性冲击。这也是为什么"效率优先"的全球化模式在面临冲击时如此脆弱。

同时，技术标准本身也具有网络效应：一旦某种标准（如USB-C、Wi-Fi）获得足够多的用户，新标准很难取代它。这就是为什么中美在技术标准上的竞争至关重要——它决定的不是哪家公司赢，而是哪个生态体系能主导未来。

给个体的启示：全球化周期带来的冲击对某些人是灾难，但对能够从中获益的人是机会。理解周期比预测方向更重要——全球化会扩张，也会收缩，但不会消失。在不确定性中建立多元连接（地理上、技能上、网络上），是最稳健的策略。
