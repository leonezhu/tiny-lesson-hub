身体与健康（生病、看医生、药店）

第6课 · ~12分钟 · 生活场景

学完本课之后，你能用四川话描述身体哪里不舒服、跟医生说明症状、在药店买药。重点是掌握四川话里独特的表达方式，而不是照搬普通话。

为什么这很重要

在成都，生病了很多人第一反应是去社区医院或者找中医馆，甚至先问亲戚推荐什么药。如果你只会说普通话，在跟本地老医生沟通时可能会觉得对方说的四川话像天书——尤其是那些描述身体感受的独特词汇。比如普通话的"发烧"，四川人说"发冷发热"或者干脆说"烧起来了"；"头疼"可以说"头昏脑涨"。

身体部位词

🏥 身体部位

  • 头壳 tóu ké — 脑袋（四川话常用"头壳"代替"头"）
  • 脑壳痛 nǎo ké tàng — 头痛，四川人常说"脑壳痛"而不是"头痛"
  • 喉咙 hóu lóng — 嗓子，四川人也说"喉管儿"
  • 胃 wèi — 同上，但四川人说"胃不舒服"更常说"胃里不得劲"
  • 肚子 dǔ zi — 腹部，四川人说"肚脐儿"
  • 背脊 bèi jǐ — 后背，四川话说"背脊骨"
  • 膝盖 xī gài — 膝盖，四川话也说"膝头儿"
  • 脚杆 jiǎo gān — 小腿/腿（四川特色词）
  • 手杆 shǒu gān — 手臂（四川特色词）
四川话特色："杆"这个后缀是四川话的一大特色。手杆（arm）、脚杆（leg/shin）、腰杆（lower back）。这些词在日常生活中非常高频，尤其是在说疼痛的时候："我手杆疼得很"。

描述症状的表达

😷 怎么说自己不舒服

  • 我发烧了 — 四川话："我烧起来了" 或 "我发冷发热"
  • 我头疼 — 四川话："我脑壳痛" 或 "我头壳痛"
  • 我咳嗽 — 四川话："我在咳" 或 "我咳得凶"
  • 我感冒了 — 四川话："我伤风了"（老派说法）或 "我感冒了"
  • 我拉肚子 — 四川话："我跑稀"（非常地道的说法！）
  • 我恶心/想吐 — 四川话："我反胃" 或 "我想呕"
  • 我胃疼 — 四川话："我胃里不得劲" 或 "我胃里头痛"
  • 我没力气 — 四川话："我好虚" 或 "我浑身没劲"
  • 我睡不好 — 四川话："我晚上睡不踏实" 或 "我失眠了"
重点词汇 — "跑稀"：这是四川话里最地道的"拉肚子"说法。"跑稀"字面意思是"跑着拉稀"，形容腹泻来得急又猛。记住这个词，你在成都看病的时候医生会觉得你很懂行。

对话：在社区医院

👨‍⚕️ 场景：在社区医院跟医生说明症状

医生： 咋个回事嘛？哪个地方不舒服？
病人： 医生，我这两天脑壳痛，喉咙也痛，还烧起来了。
医生： 来，张口让我看一下。嗯……扁桃体有点肿。你还有啥子症状？
病人： 就是咳，咳得凶，晚上睡不踏实。还有点跑稀。
医生： 你这是感冒加肠胃炎。我给你开点药，一天吃三次，吃完再来复查。
病人： 好嘞，谢谢医生。
对话要点：注意医生的用词——"咋个回事嘛"（怎么回事）、"哪个地方"（哪里）、"啥子"（什么）、"凶"（厉害）。这些都是四川话问诊时的典型用词。

对话：在药店买药

💊 场景：在药店跟店员说你需要什么药

顾客： 老板，我脑壳痛、喉咙痛，有啥子药可以推荐？
店员： 你烧起没有嘛？
顾客： 有一点，38度左右。
店员： 那你要得吃退烧药。这个氨酚黄那敏片，一天三次，一次两片。再配一盒板蓝根冲剂，对喉咙好。
顾客： 好，就按你说的来。多少钱？
店员： 一共四十五块。
对话要点："有啥子药"（有什么药）、"你要得吃"（你需要吃）、"就按你说的来"（就这样办）。四川话在口语中经常省略主语，直接说动词。

四川话 vs 普通话：关键区别

场景 普通话 四川话
头痛 我头痛 我脑壳痛 / 我头壳痛
发烧 我发烧了 我烧起来了 / 我发冷发热
拉肚子 我拉肚子 我跑稀
哪里不舒服 哪个地方不舒服 哪个地方不得劲 / 哪个地方痛
很厉害 很严重 很凶 / 重得很
需要吃药 你需要吃药 你要得吃药
手臂疼 我手臂疼 我手杆痛
腿疼 我腿疼 我脚杆痛

药店常见药品名（四川话语境）

💊 你可能听到的药名

  • 布洛芬 bù luò fēn — 止痛退烧，四川人说"吃颗布洛芬"
  • 板蓝根 bǎn lán gēn — 清热解毒，四川人家家常备
  • 双黄连 shuāng huáng lián — 感冒清热
  • 蒙脱石散 méng tuō shí sàn — 治拉肚子的
  • 奥美拉唑 ào měi lā zuò — 胃药
  • 云南白药 yún nán bǎi yào — 跌打损伤，四川人也很常用
  • 藿香正气水 huò xiāng zhèng qì shuǐ — 中暑、肠胃不适，味道苦但管用
实用提示："藿香正气水"是四川人家里的万能药，夏天防暑、冬天治肠胃都靠它。虽然味道苦到让人皱眉，但四川老人会说："苦是苦，管用嘛！"
