数理逻辑基础：推理的骨架

Lesson 9 · ~12 minutes · 数学与逻辑 · 核心基础

本课结束时，你将能够区分命题逻辑和谓词逻辑，读懂基本推理规则（如假言推理），理解真值表的工作原理，并且知道哥德尔不完备定理到底在说什么——以及为什么它证明了数学有内在的极限。

什么是数理逻辑？

数理逻辑（Mathematical Logic）不是"用数学方法学逻辑"，而是把逻辑本身变成一门数学。它用符号和公式代替自然语言，用严格的推导规则代替直觉判断。这样做的好处是：消除了歧义——在自然语言中，"如果明天下雨，我就不出门"可以有多种解读方式；但在命题逻辑中，"P → Q"只有一个确定的含义。

数理逻辑分为四大分支：命题逻辑（propositional logic）、谓词逻辑（predicate logic）、模型论（model theory）和证明论（proof theory）。我们这节课聚焦前两个——它们是理解一切高级逻辑的基础。

命题逻辑：最简单的推理单元

命题逻辑处理的是命题（有真值的陈述句）之间的关系。一个命题要么是"真"（T），要么是"假"（F），没有中间状态。命题逻辑用五个基本运算符连接命题：

五大逻辑运算符

  • 否定（¬）：¬P = "P 不成立"。P 为真则 ¬P 为假，反之亦然。
  • 合取（∧）：P ∧ Q = "P 且 Q"。两者都为真时才为真。
  • 析取（∨）：P ∨ Q = "P 或 Q"。至少一个为真即为真。
  • 蕴含（→）：P → Q = "如果 P，那么 Q"。仅当 P 真 Q 假时为假。
  • 等价（↔）：P ↔ Q = "P 当且仅当 Q"。两者同真或同假时为真。

最容易困惑的一行：P 为假时，P → Q 总是为真。这叫"空真"（vacuous truth）——"如果太阳从西边出来，我就是世界首富"这句话逻辑上是真的，因为前提永远不成立，你无法证伪它。

中文母语者的陷阱：自然语言中的"如果……就……"往往带有因果关系（"如果你不努力，就会失败"），但逻辑蕴含 P → Q 只关心真假值，不关心因果。"如果 2+2=5，那么雪是黑的"在命题逻辑中是真的（因为前件为假），虽然在自然语言中听起来荒谬——这正是为什么要用形式化语言代替自然语言的原因。

推理规则：从前提到结论的桥梁

命题逻辑不只是静态的真值表——它有一套推理规则，让你能从已知为真的前提，一步步推导出新的真结论。最核心的规则是假言推理（Modus Ponens）：

假言推理（Modus Ponens）

P → Q   （如果 P，则 Q）
P       （P 成立）
———————
∴ Q    （因此 Q 成立）

这是数学证明中最常用的推理规则。它的直观含义是："如果 P 足以推出 Q，而且 P 确实成立了，那么 Q 必然成立。"

其他常用推理规则

  • 拒取式（Modus Tollens）：P → Q，¬Q → ¬P（如果 Q 不成立，则 P 也不成立）
  • 假言三段论：P → Q，Q → R → P → R（传递性）
  • 构造性二难：P ∨ Q，P → R，Q → R → R（无论哪种情况，R 都成立）
  • 反证法（Reductio ad Absurdum）：假设 ¬P，推出矛盾 → 因此 P 成立

经典案例：反证法的力量

古希腊数学家 Euclid 在公元前 300 年左右证明了"素数有无穷多个"，用的就是反证法：

  • 假设素数只有有限个：p₁, p₂, …, pₙ
  • 构造一个新数 N = p₁ × p₂ × … × pₙ + 1
  • N 除以任何一个 pᵢ 都余 1，所以 N 不被任何已知素数整除
  • N 要么本身是素数，要么有不在列表中的素因子
  • 这与"素数只有有限个"矛盾 → 假设不成立 → 素数无穷多

这个证明的精妙之处在于：它没有构造出所有素数，只是证明了"不可能列出所有素数"。这是一种"存在性证明"——你不需要知道答案是什么，只需要证明答案不可能不存在。

谓词逻辑：进入命题的内部

命题逻辑有一个致命缺陷：它把每个陈述都当作一个不可分割的原子。在命题逻辑中，"苏格拉底是人"和"柏拉图是人"是两个完全无关的命题 P 和 Q。但显然，它们共享一个结构——"某人是人"。

谓词逻辑（Predicate Logic，又称一阶逻辑 First-Order Logic）引入了量词来解决这个问题：

两个核心量词

  • 全称量词 ∀：∀x P(x) = "对所有 x，P(x) 成立"。例如：∀x (人(x) → 会死(x)) = "所有人都注定会死"
  • 存在量词 ∃：∃x P(x) = "存在某个 x，使得 P(x) 成立"。例如：∃x (人(x) ∧ 哲学家(x)) = "存在既是人又是哲学家的人"

为什么谓词逻辑比命题逻辑强？

考虑这个经典三段论：

  • 前提 1：所有人都会死（∀x (人(x) → 会死(x))）
  • 前提 2：苏格拉底是人（人(苏格拉底)）
  • 结论：苏格拉底会死（会死(苏格拉底)）

在命题逻辑中，这三个句子是三个独立的原子 P、Q、R，无法推导。在谓词逻辑中，你可以用全称实例化（从 ∀x 得到具体 x）和假言推理两步就推出结论。这就是谓词逻辑的威力——它分析了命题的内部结构。

常见混淆：∀ 和 ∃ 的否定：¬∀x P(x) ≠ ∀x ¬P(x)。正确的转换是：¬∀x P(x) ↔ ∃x ¬P(x)（"并非所有人都达标" = "存在有人不达标"）。同样，¬∃x P(x) ↔ ∀x ¬P(x)（"不存在完美的人" = "所有人都有缺点"）。这在编程和 SQL 查询中极为常见。

可靠性与完全性：形式系统的完美梦想

一个形式逻辑系统有两个黄金标准：可靠性（soundness）和完全性（completeness）。

可靠性 vs 完全性

  • 可靠性：所有能证明的都是真的。如果系统证明了 P，那 P 在现实中一定为真。这是"不犯错"的保证。
  • 完全性：所有真的都能证明。如果 P 在现实中为真，那系统一定能找到一条证明路径。这是"无所不能"的承诺。

一个完美的系统应该同时满足两者：能证明的 = 真的。1900 年代初，数学家 Hilbert 提出了"形式化全部数学"的计划，期望建立一个既可靠又完全的数学基础。

哥德尔不完备定理：梦想的破灭

1931 年，25 岁的 Kurt Gödel 发表了一篇 46 页的论文，彻底粉碎了 Hilbert 的梦想。他的不完备定理说的是：

哥德尔第一不完备定理

任何足够强大的形式系统（能表达基本算术），都存在一个命题 P，使得 P 为真，但 P 在系统内无法被证明。

换句话说：真理 > 可证明性。有些东西是真的，但你永远无法在系统内证明它。

哥德尔的证明思路（简化版）

Gödel 做了一个天才的操作：他把逻辑命题"编码"成了数字（哥德尔编号），然后构造了一个特殊的数学命题 G，其含义是：

  • "G 在这个系统中不可证明"

现在问：G 是真的还是假的？

  • 如果 G 是假的 → "G 不可证明"是假的 → G 是可证明的 → 但系统如果是可靠的，可证明的就是真的 → G 应该是真的 → 矛盾！
  • 如果 G 是真的 → "G 不可证明"是真的 → G 确实不可证明 → 一致，没有矛盾。

所以 G 一定是真的——但 G 在系统内不可证明。这就是不完备定理的核心洞见。

最重要的澄清：哥德尔不完备定理不是说"数学有错误"——它不是说系统会推导出矛盾的结论。它说的是：任何足够强的系统，都有一些"真命题"永远无法在系统内被证明。这不代表数学"不够好"——它代表数学的"真"和"可证明"是两个不同的概念。这也意味着，不存在一个通用算法能自动判断所有数学命题的真假。

总结：从命题到哥德尔

我们今天走过了数理逻辑的核心脉络：

  • 命题逻辑——用符号表示简单的真值关系，用推理规则从前提推出结论
  • 谓词逻辑——进入命题内部，用量词表达"所有"和"存在"，让三段论等形式推理成为可能
  • 可靠性与完全性——形式系统追求的两大黄金标准
  • 哥德尔不完备定理——任何足够强的系统都有内在极限，真理永远超出可证明的范围

这不仅仅是数学知识。理解数理逻辑，让你在面对 AI 输出、科学论证、甚至日常争论时，能分辨"形式上有效的推理"和"实质上正确的结论"——这是批判性思维的最高级别。
