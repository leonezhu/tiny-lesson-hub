信息论基础 — 速查卡

来自 Lesson 6 · 快速参考

核心直觉

信息的多少 = 意外的程度

  • 高概率事件 → 低信息量
  • 低概率事件 → 高信息量
  • 确定性事件（概率=1）→ 零信息
香农熵
H(X) = −∑ P(xi) · log2(P(xi))

熵的直觉

  • 熵 = 不确定性的度量
  • 熵 = 消除不确定性所需的信息量
  • 熵 = 无损压缩的理论极限
  • 熵越高，越不可预测；熵越低，越有规律

常见熵值

事件 概率 信息量(bits)
公平硬币正面 0.5 1.00
掷骰子是6 1/6 2.58
从100选1 0.01 6.64
确定性事件 1.0 0

霍夫曼编码原理

  • 高频字符 → 短码（少用bits）
  • 低频字符 → 长码（多用bits）
  • 平均编码长度逼近熵值
  • 不能压缩的数据 = 最大熵的随机数据

四大应用

  • 通信：信道容量定理 → 5G/Wi-Fi/光纤
  • 加密：高熵密文 = 不可预测
  • AI：信息增益 → 决策树；交叉熵 → 损失函数
  • 数据科学：模式（低熵）vs 噪声（高熵）
