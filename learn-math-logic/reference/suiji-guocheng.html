随机过程速查

参考卡 · 数学与逻辑 Lesson 11

概率论动态系统跨学科

一句话定义

随机过程 = 随时间演化的随机变量序列。不是描述一个确定性的世界，而是描述概率如何随时间演变。

三大核心类型

随机游走（Random Walk）

St = S0 + Σ Xi，典型位移 ∝ √t
  • 每一步独立，方向随机，无记忆
  • 扩散律：n 步后距离 ≈ √n
  • 应用：股价模型、布朗运动、赌博分析

马尔可夫链（Markov Chain）

P(Xt+1 | Xt, ..., X0) = P(Xt+1 | Xt)
  • 无记忆性：下一步只取决于当前状态
  • 由状态空间 + 转移矩阵定义
  • 可收敛到稳态分布
  • 应用：PageRank、RL(MDP)、语言模型、排队论

泊松过程（Poisson Process）

P(N(t) = k) = (λt)k e-λt / k!
  • 描述"单位时间内事件发生次数"
  • 事件相互独立，到达率 λ 恒定
  • 应用：排队系统、核衰变、网络流量建模

关键数字

  • Bachelier (1900) — 首次用随机游走描述股价
  • Einstein (1905) — 用布朗运动证明原子存在
  • Markov (1907) — 提出马尔可夫链，分析普希金诗歌
  • Kolmogorov (1933) — 概率论公理化
  • Black-Scholes (1973) — 几何布朗运动给期权定价
  • PageRank (1998) — 马尔可夫链改变信息检索

常见误区

赌徒谬误："已经连续 5 次正面，下次必然是反面"——这是对随机游走无记忆性的根本误解。每次投掷都是独立的。
肥尾陷阱：正态分布假设极端事件极其罕见，但 Mandelbrot 发现金融数据的尾部远厚于正态——极端事件比模型预测的多得多。
