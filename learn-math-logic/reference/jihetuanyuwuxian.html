核心概念

集合（Set）：元素的容器。集合论研究集合之间的关系与运算。

基数（Cardinality）：集合"大小"的度量。两个集合基数相等 ↔ 存在一一对应（双射）。

一一对应（Bijection）：两个集合元素一一配对，不多不少。这是比较无穷集合大小的唯一可靠方法。

关键结果

  • N（自然数） 可数，基数 = ℵ₀（阿列夫零）
  • Z（整数） 可数，基数 = ℵ₀ — 可以枚举为 0, 1, -1, 2, -2, …
  • Q（有理数） 可数，基数 = ℵ₀ — 可以按分子分母之和枚举
  • R（实数） 不可数，基数 = 2^ℵ₀ > ℵ₀
  • Cantor 定理：|P(S)| > |S| 对任何集合 S 成立

对角线论证（核心思路）

假设 (0,1) 中的实数可枚举为 r₁, r₂, r₃, …
构造 x：第 n 位 ≠ rₙ 的第 n 位
→ x ≠ 列表中任一 rₙ → 矛盾
→ (0,1) 不可数

无穷阶梯

ℵ₀ < 2^ℵ₀ < 2^(2^ℵ₀) < …
没有最大的无穷

常见误解

  • ❌ "无穷很大" → ✅ 无穷是一种结构类型，不是"非常大"
  • ❌ "有理数比整数多" → ✅ 它们一样大（都是 ℵ₀）
  • ❌ "无限集合不能和子集一样大" → ✅ 无限集合可以和真子集等势
