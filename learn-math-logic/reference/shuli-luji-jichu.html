五大逻辑运算符

  • 否定 ¬P — "P 不成立"
  • 合取 P ∧ Q — "P 且 Q"
  • 析取 P ∨ Q — "P 或 Q"
  • 蕴含 P → Q — "如果 P，那么 Q"（仅 P真Q假时为假）
  • 等价 P ↔ Q — "P 当且仅当 Q"

核心推理规则

  • 假言推理：P → Q, P ∴ Q
  • 拒取式：P → Q, ¬Q ∴ ¬P
  • 假言三段论：P → Q, Q → R ∴ P → R
  • 反证法：假设 ¬P → 矛盾 ∴ P

谓词逻辑量词

  • ∀x P(x) — "对所有 x，P(x) 成立"
  • ∃x P(x) — "存在 x，P(x) 成立"
  • ¬∀x P(x) ↔ ∃x ¬P(x) — "并非所有" = "存在不"
  • ¬∃x P(x) ↔ ∀x ¬P(x) — "不存在" = "所有不"

可靠性与完全性

  • 可靠性：能证明的 → 真的（不犯错）
  • 完全性：真的 → 能证明（不遗漏）
  • 哥德尔不完备定理：任何足够强的系统，存在真但不可证的命题
