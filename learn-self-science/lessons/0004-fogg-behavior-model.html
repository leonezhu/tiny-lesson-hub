第四课：B=MAP行为设计模型

自我科学与行为改变 · Lesson 4 of 12

你以前肯定听过这句话："只要你想，你就一定能做到。"但事实是——你不想，或者你试了就是做不到。斯坦福大学行为设计师 BJ Fogg 发现了一个更诚实的公式：B = MAP。它告诉你：行为不是由"想不想"决定的，而是由三个因素同时满足决定的。Motivation（动机）、Ability（能力）、Prompt（提示）。

🏠 生活场景
你是不是也这样——晚上躺床上发誓明天一定要早起跑步，结果闹钟响了按掉继续睡？第二天早上 Motivation 爆棚（"我要瘦！"），但 Ability 为零（困得要死），Prompt 也不存在（床太舒服了）。结果呢？行为 = 0。

这不是你意志力差。这是 B=MAP 公式在起作用。

这节课我们来拆解这个公式，然后学会用它诊断自己的每一个"做不到"。

一、B=MAP 公式拆解

Fogg 的行为模型可以用一个极简的公式表示：

Fogg Behavior Model
B = M × A × P
Motivation × Ability × Prompt = Behavior

三个要素缺一不可。任何一个为零，行为就不会发生。我们来逐个看。

1.1 Motivation（动机）— "我想做"

Motivation 是你想做事的欲望强度。Fogg 发现，动机不是固定不变的——它在一天之内波动巨大。

💡 关键洞察
你的动机像一个钟摆。早上你想健身，中午你想学习，晚上只想刷手机。这不是你"不自律"，这是人类动机的自然规律。

动机有两个方向：

正向动机：追求快乐 / 希望
  • 追求快乐 — 吃美食、打游戏、刷短视频带来的即时快感
  • 寻求希望 — 想到"学了英语可以去更好的公司"而开始背单词
  • 避免痛苦 — 怕胖所以少吃（这是恐惧驱动）
  • 避免丧失 — 怕错过限时优惠而下单
负向动机：逃避痛苦 / 恐惧
  • 逃避痛苦 — 牙疼去看医生
  • 避免丧失 — 怕丢工作所以加班
  • 社会压力 — 别人都买了你也买

但问题来了：动机是最不可靠的行为触发因素。为什么？因为它像电量——用完了就得充，而且充得很慢。

🏠 生活场景
你刚加完班，累得像条狗。这时候你说"我要去跑步"——动机为零。就算你有再强的"自律精神"，身体也不会听你的。这不是道德问题，这是生物学问题。

1.2 Ability（能力）— "我能做"

Ability 不是你有没有"能力"，而是这件事对你来说有多容易。Fogg 强调：不要相信"我做不到是因为我不会"，大多数时候是"我做不到是因为太难了"。

影响 Ability 的六个因素（从最重要到最不重要）：

因素说明例子
时间需要做多少事？"花2小时写报告" vs "花30秒回复消息"
金钱需要花多少钱？买一杯奶茶 vs 买一辆车
体力需要多少体力？走一步 vs 跑五公里
脑力需要多少思考？打开微信 vs 解微积分题
社会偏差别人怎么看？在办公室唱歌 vs 在家里唱歌
非常规性是不是打破常规？正常走路 vs 倒着走
💡 核心原则
想让行为更容易发生？别去提高动机，先去降低难度。 这就像减肥——与其逼自己饿肚子（提高痛苦动机），不如把零食扔掉（降低获取难度）。

1.3 Prompt（提示）— "现在做"

Prompt 是触发行为的信号。没有提示，再强的动机和再高的能力都不会转化为行为。

Fogg 区分了三种提示：

✅ 火花型提示（Spark Prompt）

触发已有动机的信号。比如看到健身广告→想起"我要瘦"→去健身。这类提示依赖动机水平，动机低时无效。

⚡ 行动型提示（Action Prompt）

直接告诉你要做什么。比如手机闹钟显示"该喝水了！"。这类提示最有效的前提是行为已经足够简单。

🔔 简单提示（Tiny Prompt）

最微小的触发信号。比如把跑鞋放在门口——看到就跑鞋=去跑步。这种提示之所以有效，是因为行为本身足够简单。

最常见的错误：在动机很低的时候，用一个微弱的提示指望引发重大行为。比如你在沙发上躺着，手机弹了一条"今天学习了吗？"的通知——然后你关掉了通知继续躺。

🔍 诊断：为什么你没学习？

  • Motivation：低（刚下班，不想动脑）
  • Ability：低（书太厚，不知道从哪开始）
  • Prompt：弱（一条通知而已）

结论：三个条件都不满足，不学习是必然结果。不是你不努力，是设计不合理。

二、动机-能力曲线（The Behavior Map）

Fogg 做了一个著名的图表，横轴是能力（从左到右越来越容易），纵轴是动机（从下到上越来越高），中间有一条分界线：

行为地图（简化版）
📈
高动机 + 难 = ❌ 不行
比如：熬夜时逼自己写论文
🎯
低动机 + 易 = ✅ 可以
比如：累了就刷5分钟书
⚡
高动机 + 易 = 💪 最好
比如：有劲时做大事

这条曲线的核心启示是：当动机低的时候，你必须让行为变得极其简单。

💡 这就是"微习惯"的科学依据
很多人以为微习惯是"懒人做法"，其实恰恰相反——微习惯是在承认人类动机的天然波动性之后，做出的最理性的策略选择。你不是在降低标准，你是在用最低成本维持行为的发生频率。

2.1 现实案例：健身房的真相

🏠 生活场景
小明办了健身卡，第一天热血沸腾，做了半小时有氧。第二天上班很累，心想"明天再去"。第三天忘了。第四天想起来，觉得"都三天没去了，下次再说"。

分析：
- 第一天：动机高 ✅，行为设计合理 ✅ → 成功了
- 第二天：动机低 ❌，但行为还是"去健身房30分钟"（太难）❌ → 失败
- 第四天：动机中等，但行为太复杂 → 放弃

如果用 B=MAP 重新设计：
- 低动机日改为"在家做5个俯卧撑"（能力↑↑）
- 高动机日再去健身房（能力↑ + 动机↑ = 💪）
- 设置固定提示：下班回家先换运动服（Prompt✅）

三、如何用 B=MAP 设计你的行为

现在你知道公式了，接下来是实操。Fogg 建议按照以下四个步骤来设计任何你想养成的行为：

3.1 第一步：选一个你想要的小行为

不是"我要变得自律"，而是具体的、可观察的行为：

❌ 模糊目标 → "我要多读书"
✅ 具体行为 → "每晚睡前读2页书"

❌ 模糊目标 → "我要更健康"
✅ 具体行为 → "每天午餐后散步10分钟"

3.2 第二步：为这个行为找到合适的时机和提示

提示必须具体、可见、不可忽略：

🏠 生活场景
你想养成"每天喝水"的习惯。
- 弱提示：心里想着"该喝水了"（容易被忘）
- 强提示：在电脑旁边放一个装满水的大杯子（视觉可见，触手可及）

关键原则：提示必须出现在行为即将发生的时间和地点。

3.3 第三步：降低行为的难度

问自己："如果我现在一点都不想做，这个行为最小可以小到什么程度？"

📖 阅读

最小行为：翻开书，读一行字。就这么小。小到你的 System 1 不会反抗。

🏃 运动

最小行为：穿上运动鞋。或者做1个深蹲。小到不可能失败。

✍️ 写作

最小行为：打开文档，写一句话。哪怕只是"今天不想写"。

3.4 第四步：设置庆祝（Celebration）

这是 Fogg 模型中最被低估的一环。完成行为后，立即给自己一个小型庆祝：

🎉 庆祝不是奖励，是神经信号的锚定
做一个深呼吸，在心里说"太棒了！"，或者比个耶。这个动作会告诉你的大脑："刚才那件事是好的，以后多做。"

庆祝不需要物质奖励。一句"好样的"就够了。因为你在训练的是大脑的情绪记忆，不是在做交易。
🏠 生活场景
你完成了"读2页书"的最小行为。合上书，对自己笑一下，说"不错嘛"。然后继续做别的事。

注意：不要说"才读了两页有什么用"。这句话会摧毁你刚刚建立的正向神经连接。庆祝的目的是让行为本身成为奖励，而不是让行为成为获得奖励的手段。

四、诊断工具：你的行为为什么没发生？

当你发现自己"又没做到"的时候，不要自责。拿出 B=MAP 的诊断表，逐项检查：

🔍 B=MAP 诊断模板

行为：[填写你想做的行为]

  • Motivation（动机）：当时我想做吗？（1-10分）___
  • Ability（能力）：这件事对我来说容易吗？（1-10分，10=非常简单）___
  • Prompt（提示）：当时有明确的触发信号吗？（有/无）___

改进方案：从三个维度中选一个最容易调整的，优先修改它。

4.1 常见场景诊断

场景1：久坐腰疼

  • Motivation：知道该起来活动，但"等会儿再说"→ 动机中等偏低
  • Ability：站起来走两步不难 → 能力尚可
  • Prompt：没有定时提醒 → 提示缺失 ❌

解决方案：设一个每45分钟的闹钟（Prompt✅），闹钟响了就做1个拉伸（Ability✅）。不需要很高动机。

场景2：刷视频逃避工作

  • Motivation：不想面对困难的工作 → 负向动机
  • Ability：刷视频极其简单 → 能力极高
  • Prompt：手机就在手边，App图标在首页 → 提示极强

解决方案：把社交 App 移到文件夹深处（增加难度↓），设一个"打开工作文档"的桌面快捷方式（Prompt✅），告诉自己"只做5分钟"（降低Ability门槛）。

场景3：健身房中断

  • Motivation：刚开始很高，一个月后下降 → 动机衰减
  • Ability：去健身房路上+换衣服+锻炼 → 综合难度高
  • Prompt：没有固定时间 → 提示不稳定

解决方案：固定每周二四六晚上8点（Prompt✅），当天只要求自己"换上运动服"（Ability↑↑），到了健身房再说（动机自然会来）。

五、动机陷阱：为什么"打鸡血"没用

我们从小被教育："只要有足够的动力，你就能做任何事。"但 Fogg 的研究推翻了这一点：

⚠️ 动机陷阱
1. 动机来得快，去得更快。靠打鸡血驱动的行为，通常坚持不超过一周。
2. 高动机时的行为设计不可靠。你在精力充沛时制定的计划（"每天跑5公里"），在疲惫时根本执行不了。
3. 动机和能力是替代关系。行为越难，需要的动机越高；行为越简单，需要的动机越低。聪明的做法是让行为简单到几乎不需要动机。
🏠 生活场景
你有没有过这种经历——凌晨两点睡不着，突然灵感爆发，列了一个"明天开始的人生计划"：早起、跑步、读书、学英语、早睡……然后第二天早上闹钟一响，全部归零。

这不是你"缺乏执行力"。这是你在低动机时段（凌晨困倦）和高动机时刻（灵感爆发）做了同一件事的设计。这两个时刻的 M-A-P 条件完全不同，但计划却一样。当然执行不了。

5.1 应对策略：为低动机日设计

Fogg 的建议是：永远按照你最低动机、最低能力的状态来设计行为。

为什么？因为：

  • 如果你的行为在低动机状态下也能完成，那么在高中机状态下一定也能完成
  • 行为的一致性比单次行为的强度更重要
  • 低动机日的成功体验会积累成自我效能感（"我能做到"的信心）
💡 "最低可行行为"（Minimum Viable Action）
在任何你想培养的行为上，问自己："如果我现在一点也不想动，我能做的最小一步是什么？"

跑步的最小一步：穿上跑鞋。
读书的最小一步：翻开书。
工作的最小一步：打开电脑。

只要做了这一步，你就赢了。 多做了是 bonus，没多做也不是失败。

七、本课总结

概念一句话记住它
B = MAP行为 = 动机 × 能力 × 提示，三者缺一不可
动机不可靠动机像电量，用完了就没了，不能靠它办事
能力 > 动机让行为变简单，比提高动机有效得多
提示要具体模糊的"想想看"不算提示，"看到跑鞋就穿"才算
最低可行行为按你最不想动的时候来设计，这样任何时候都能做
庆祝很重要完成行为后立即给自己一个小肯定，这是在训练大脑
