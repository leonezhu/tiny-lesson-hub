Passive Voice — Quick Reference

From Lesson 9: Passive Voice — When & Why

Formula

be + past participle (V3) — the "be" verb changes to match the tense.

Passive Across Tenses

TenseActivePassive
Present SimpleShe writes the report.The report is written.
Past SimpleShe wrote the report.The report was written.
Present PerfectShe has written the report.The report has been written.
Future (will)She will write the report.The report will be written.
Modal (should)She should write the report.The report should be written.
Present Cont.She is writing the report.The report is being written.

When to Use

✓ Unknown doer: "My wallet was stolen."
✓ Irrelevant doer: "The policy was changed."
✓ Focus on recipient: "The CEO was interviewed."
✓ Scientific/formal: "The sample was heated to 100°C."

When to Avoid

✗ You know the doer and want to credit/blame: Use active — "We fixed the bug."
✗ Personal communication: Emails, updates, Slack — use active.
✗ When clarity matters: Passive can obscure who is responsible.
