Conditionals B2 Grammar

Quick reference card · Full Lesson

Overview

Type If-clause Main clause Use
Zero present simple present simple General truths, facts
First present simple will + base verb Real future possibilities
Second past simple would + base verb Unreal / unlikely present or future
Third past perfect would have + past participle Unreal past (regrets, criticism)
Mixed past perfect would + base verb Past cause → present result

Zero Facts

If + present, present

  • If you heat water to 100°C, it boils.

First Real Future

If + present, will + base

  • If it rains tomorrow, I'll bring an umbrella.

Second Unreal Present/Future

If + past, would + base

  • If I had more time, I would learn guitar.

Third Unreal Past

If + past perfect, would have + past participle

  • If she had read the email, she would have met the deadline.

Mixed Past → Present

If + past perfect, would + base

  • If I had studied engineering, I would be at Google now.

Key Patterns

  • Unless = if ... not: "Unless you apologize, I won't forgive you."
  • Were I you = If I were you (formal, inversion)
  • Provided that / as long as = only if
  • Implied if: "I would help, but I'm busy."
