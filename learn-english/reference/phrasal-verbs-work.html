Quick Reference: Phrasal Verbs at Work

12 phrasal verbs from Lesson 8 · 可分离 vs 不可分离

Phrasal Verb Meaning Separable? Example
bring up 提出（话题） Yes Bring it up in the meeting.
follow up 跟进 No (with on/with) Follow up with the client.
wrap up 完成、总结 Yes Wrap it up by EOD.
put off 推迟 Yes Put it off until tomorrow.
run into 遇到（问题/人） No We ran into a problem.
look into 调查、研究 No Look into the error logs.
set up 设置、建立 Yes Set it up in the calendar.
figure out 搞清楚、解决 Yes Figure it out before stand-up.
point out 指出 Yes Point it out in the review.
cut down on 减少 Yes Cut down on unnecessary meetings.
keep up with 跟上（进度） No Keep up with the sprint.
touch base 联系、碰头 No Let's touch base after the demo.

Key Rule

Separable phrasal verbs (7): With pronouns (it/them), the pronoun MUST go between the verb and particle: "figure it out" ✅ / "figure out it" ❌

Not separable phrasal verbs (5): The object always comes after the full phrase: "run into it" ✅ / "run it into" ❌
