RLHF 与人类对齐 — 快速参考卡

AI 第十课 · 核心概念速查

三阶段训练路径

预训练 → 读万卷书（吸收知识）

SFT → 学会格式（懂规矩、听指令）

RLHF → 学会好坏（有情商、有价值观）

RLHF 三步机制

1. 奖励模型：同一 prompt 生成 4–16 个回答 → 人类排序 → 训练打分模型

2. PPO 微调：R(θ) − β·KL(π_θ || π_ref)，KL 约束防止模型走捷径

3. 迭代循环：部署 → 收集反馈 → 更新奖励模型 → 重新训练

关键数据

  • InstructGPT：~500 名标注者、~1 万条示范样本
  • RLHF 后：有害性减少 ~40%，有用性提高 ~30%
  • 人类标注一致性：60–80%
  • β (KL 权重) ≈ 0.03
  • 越狱成功率：10–30%

DPO vs RLHF

RLHF：训练 SFT + 奖励模型 + PPO 微调。灵活但复杂。

DPO：直接对比好/坏答案，无需奖励模型和 PPO。简单高效，开源主流。

核心局限

  • 人类共识上限：奖励模型无法超越标注者群体的一致性
  • 奖励黑客：模型学会表面服从而非真正对齐
  • 文化偏差：标注者群体的价值观 ≠ 普世价值观
  • 目标设定：奖励模型捕捉偏好模式 ≠ 完整人类意图
