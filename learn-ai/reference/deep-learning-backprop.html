深度学习与反向传播

快速参考卡 · 第三课概要

核心公式

权重更新
w_new = w_old - learning_rate · ∂L/∂w
链式法则（反向传播核心）
∂L/∂w₁ = ∂L/∂ŷ · ∂ŷ/∂h · ∂h/∂w₁
常见损失函数
MSE: L = (1/n) Σ(ŷᵢ - yᵢ)²   （回归）
Cross-Entropy: L = -Σ yᵢ log(ŷᵢ)   （分类）

反向传播三步走

  • 前向传播：输入 → 网络 → 输出 → 计算损失
  • 反向传播梯度：从输出层往回，链式法则计算 ∂L/∂w 对每个参数
  • 更新权重：w = w - lr · ∂L/∂w

超参数速查

学习率

  • 太大 → 跳过最低点、发散
  • 太小 → 训练极慢
  • 典型范围：0.001 ~ 0.01（Adam）

批量大小

  • 小（16-32）：噪音大但泛化好
  • 中（128-256）：平衡之选
  • 大（1024+）：效率高但可能泛化差

优化器对比

  • SGD：简单，统一学习率，对超参数敏感
  • SGD + Momentum：加入速度，穿过平坦区域更快
  • Adam：自适应学习率，默认选择，lr=0.001
  • AdamW：Adam + 独立权重衰减，通常更好

过拟合防治清单

  • 数据增强：翻转、裁剪、加噪声
  • Dropout：训练时随机关神经元
  • 早停：验证集性能不升即停
  • 权重衰减（L2）：惩罚大权重

激活函数速查

  • ReLU: f(x) = max(0, x) — 隐藏层首选
  • Sigmoid: (0, 1) — 二分类输出层
  • Tanh: (-1, 1) — 有时用于 RNN
  • Softmax: 概率分布 — 多分类输出层
