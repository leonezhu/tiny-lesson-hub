人工神经元

y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)

输入 × 权重 + 偏置 → 激活函数 → 输出

四个组成部分

  • 输入 (x)——来自上一层的信号
  • 权重 (w)——每个输入的重要性（学到的知识）
  • 偏置 (b)——神经元的激活门槛
  • 激活函数 (f)——引入非线性

常用激活函数

  • ReLU：f(x) = max(0, x) — 最常用，高效，缓解梯度消失
  • Sigmoid：f(x) = 1/(1+e⁻ˣ) — 输出 (0,1)，用于二分类输出层
  • Tanh：f(x) = (eˣ - e⁻ˣ)/(eˣ + e⁻ˣ) — 输出 (-1,1)，以 0 为中心
  • Softmax：输出概率分布（和为 1），用于多分类输出层

XOR 问题

单层感知机无法解决的经典问题：两类数据在平面上呈对角分布，无法用一条直线分开。证明了解决方案必须是非线性的——需要隐藏层。

前向传播流程

  • 输入层 → 接收原始数据
  • 隐藏层 → 每层做加权求和 + 激活，逐层提取高级特征
  • 输出层 → 产生最终预测

万能近似定理

一个带有至少一个隐藏层的前馈神经网络，只要隐藏层有足够多的神经元，就可以以任意精度逼近任何连续函数。

损失函数

  • MSE（均方误差）——回归问题：衡量预测值与真实值的平方差
  • 交叉熵——分类问题：衡量预测概率分布与真实标签的差异

关键人物

  • M McCulloch & W Pitts (1943) — 第一个人工神经元模型
  • Frank Rosenblatt (1958) — 感知机与学习规则
  • Minsky & Papert (1969) — 《Perceptrons》，揭示单层局限
  • Rumelhart et al. (1986) — 多层反向传播的实践证明
  • Geoffrey Hinton (2006+) — 深度学习革命
