三个训练阶段

1. 预训练（Pre-training）

目标：预测下一个 token
数据：数千亿到十万亿 token
学到：语法、知识、推理模式、世界模型

2. 监督微调（SFT）

目标：学会遵循指令
数据：人工编写的高质量指令-回答对
效果：从"续写文本"变为"回答问题"

3. 人类反馈对齐（Alignment）

RLHF：奖励模型 + PPO 强化学习（ChatGPT 用）
DPO：直接偏好优化，绕奖励模型（Claude 用）
目标：让输出符合人类偏好

关键概念

涌现能力（Emergent Abilities）

模型规模跨越阈值后突然出现的能力：
算术、指令跟随、少样本学习、代码生成

→ 不是魔法，是大规模统计学习的自然结果

缩放定律（Scaling Laws）

Kaplan (2020)：L ∝ C-0.07，计算量越大越好
Chinchilla (2022)：参数量与数据量应同步增长

关键事实

  • LLM 不是知识库，是概率生成模型
  • 回答"地球到月球距离" = 统计匹配 + 概率生成，不是查表
  • GPT = Decoder-only，BERT = Encoder-only
  • LLM 会"幻觉"——生成看似合理但错误的事实
  • 能力边界：擅长生成/理解，不擅长保证准确

模型规模对比

  • GPT-3：1750 亿参数，5700 亿 token
  • PaLM：5400 亿参数，7800 亿 token
  • GPT-4：估计数万亿参数，10 万亿+ token
