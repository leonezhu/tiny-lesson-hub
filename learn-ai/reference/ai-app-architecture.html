多模态架构

四层结构

  • 输入编码器：文本/图像/音频 → 向量表示（CLIP 等）
  • 融合层：跨模态对齐，共享语义空间
  • 统一模型：融合后进行推理（GPT-4V 等）
  • 输出解码器：向量 → 人类可读形式

Agent 系统架构

核心组件

  • 规划器：大任务 → 子步骤
  • 记忆系统：短期上下文 + 长期知识库
  • 工具调度器：决定调用哪个外部工具
  • 反思模块：检查结果，重新规划

Agent vs 聊天机器人

聊天机器人：接收问题 → 生成回答（单向）
Agent：感知 → 规划 → 行动 → 观察 → 调整（闭环决策）

企业部署架构

部署模式对比

  • MaaS：API 调用，最快上线，数据出域
  • 私有化部署：本地运行，数据不出域，需要 GPU
  • 混合架构：简单用云端，敏感用本地（主流）
  • 模型路由：按复杂度选模型，成本最优

关键指标

  • 延迟：实时 <200ms，批量 <5s
  • 吞吐量：数千至数万 QPS
  • 可靠性：降级策略、熔断、自动重试
  • 安全：加密、访问控制、审计日志
