三大核心组件

1. 卷积层（Convolution）

用滤波器（核 kernel）在图片上滑动，检测局部模式。

  • 3×3 滤波器 = 9 个权重参数
  • 浅层学低级特征（边缘、颜色）
  • 深层学高级语义（猫脸、车轮）
  • 参数共享：同一滤波器在整个图上复用

2. 池化层（Pooling）

降采样，缩小尺寸，保留关键信息。

  • 最大池化（Max Pooling）：取窗口内的最大值
  • 2×2 窗口 + 步长 2 → 尺寸变为 1/4
  • 带来平移不变性

3. 全连接层（Fully Connected）

综合所有特征，输出最终分类。

  • 位于 CNN 末端
  • 输入是展平的特征向量
  • 输出类别概率（softmax）

关键公式

输出尺寸 = (输入尺寸 − 滤波器尺寸) / 步长 + 1
例：(224 − 3) / 1 + 1 = 222（无填充，步长 1）

池化降采样：2×2 + 步长 2 → 输出 = 输入 / 2

CNN 的三大优势

  • 参数共享：同个滤波器复用，参数量大幅减少
  • 局部连接：每个神经元只连一小块区域，符合图像空间结构
  • 平移不变性：通过池化，物体在图中的位置不影响特征提取

典型架构

  • 输入（224×224×3）
  • 卷积 → 激活（ReLU）→ 池化
  • 卷积 → 激活 → 池化
  • …（更多层）
  • 展平（Flatten）
  • 全连接 → Softmax → 分类结果

经典模型

  • LeNet (1998)：手写数字识别，CNN 的开山之作
  • AlexNet (2012)：ImageNet 竞赛冠军，深度学习爆发的起点
  • VGGNet (2014)：用 3×3 小滤波器堆叠，证明深度是关键
  • ResNet (2015)：残差连接解决训练深层网络的梯度消失问题
  • EfficientNet (2019)：自动搜索最优的网络宽度、深度、分辨率组合

应用场景

  • 人脸识别、图像分类、目标检测
  • 医疗影像分析（CT/MRI 肿瘤检测）
  • 自动驾驶环境感知
  • 工业缺陷检测
  • 图像生成（GAN / Diffusion 的基础）
