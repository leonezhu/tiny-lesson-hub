RNN 与序列模型

快速参考卡 · AI 课程第五课

核心问题

普通神经网络（CNN、MLP）的输入是独立的、固定的。但真实世界很多数据是序列：文本、语音、时间序列、视频——顺序至关重要。

RNN 的核心机制

隐藏状态（Hidden State）— 网络的"记忆"

ht = tanh(Whhht-1 + Wxhxt)

yt = Whyht

在每个时间步 t，RNN 用当前输入 xt 和之前隐藏状态 ht-1 更新记忆，然后产生输出。

三种架构

单向 RNN 从前向后处理（适合时间序列预测）

双向 RNN 同时从前向后和从后向前（适合文本理解，需要完整上下文）

Encoder-Decoder 编码输入 → 压缩向量 → 解码生成输出（机器翻译、文本摘要）

长期依赖问题

梯度消失：长序列中误差信号逐层衰减，网络"忘记"早期信息。

比喻：传话游戏——信息经过多人传递后严重失真。

LSTM 的三个门

遗忘门 ft = sigmoid(...) → 决定遗忘什么

输入门 it + C̃t → 决定记住什么新信息

输出门 ot → 决定输出什么

关键：细胞状态（Cell State）是一条贯穿所有时间步的"高速公路"，信息可以几乎无损地传递。

GRU（简化版 LSTM）

更新门 — 合并遗忘门和输入门

重置门 — 控制旧信息如何被新信息覆盖

参数更少、训练更快，效果与 LSTM 相近。

典型应用

  • 机器翻译、文本生成、情感分析
  • 语音识别、音乐生成
  • 股票预测、蛋白质结构分析

与前后的关系

RNN 的长期依赖问题 → 催生了注意力机制（Attention） → 构成了 Transformer 的核心 → 开启了 LLM 时代。
