神经网络基础

第二课 · ~10 分钟 · 神经网络入门

学完这节课后，你将能够：
1. 画出一个人工神经元的基本结构并解释每个组成部分的作用
2. 解释为什么单层感知机无法解决 XOR 问题
3. 描述前向传播的计算过程
4. 理解"隐藏层"为什么是神经网络突破线性限制的关键

从生物神经元到人工神经元

要理解神经网络，首先要理解它的灵感来源——生物大脑中的一个基本单位：神经元。人脑大约有 860 亿个神经元，每个神经元通过数千个突触与其他神经元相连。一个生物神经元的工作方式可以简化为三个步骤：

生物神经元的工作流程

  • 接收信号——树突（dendrites）从其他神经元接收电信号
  • 整合信号——细胞体将所有输入信号加权求和，如果总和超过某个阈值，就"点火"
  • 传递信号——轴突（axon）将脉冲传递给下一个神经元的树突

1943 年，数学家 Warren McCulloch 和神经学家 Walter Pitts 做了一个大胆的想法：如果我用一个简单的数学公式来模拟这个流程呢？他们提出了第一个"人工神经元"模型——这个模型极其简单，却构成了整个深度学习的基石。

输入层 x₁ x₂ x₃ → → 加权求和 + 激活 f → → 输出 y
y = f(w₁x₁ + w₂x₂ + w₃x₃ + b)

人工神经元的四个组成部分

  • 输入 (x)——来自上一层的信号，可以是像素值、文本编码、或者任何其他数值
  • 权重 (w)——每个输入的重要性。权重越大，该输入对输出的影响越大。权重就是神经网络"学到的知识"
  • 偏置 (b)——一个额外的偏移量，决定神经元在什么条件下"更容易激活"
  • 激活函数 (f)——一个非线性函数，决定是否"点火"以及输出多少。没有它，神经网络就只是一堆线性变换的组合
类比理解：想象你在评估是否接受一份工作邀请。每个因素（薪资、地点、发展）都有一个权重（对你有多重要），还有一个偏置（你内心的最低接受标准）。把所有因素加权求和，再通过你的"决策函数"（比如：总分超过 70 就接受），你就做出了决定——这就是一个神经元的计算过程。

激活函数——神经网络的"开关"

如果只有加权求和，神经网络本质上就是一个线性回归模型——无论有多少层，它永远只能学习线性关系。激活函数的引入打破了这个限制，让神经网络可以学习任意复杂的非线性关系。

常见的激活函数

  • Sigmoid——将任意实数压缩到 (0, 1) 之间。早期神经网络常用，但现在在隐藏层中较少使用，因为它容易导致"梯度消失"问题。
  • Tanh——将实数压缩到 (-1, 1) 之间，以 0 为中心。比 Sigmoid 好一些，但同样存在梯度消失问题。
  • ReLU（Rectified Linear Unit）——f(x) = max(0, x)。大于 0 的值保持不变，小于 0 的值变为 0。这是目前最常用的激活函数——简单、高效、不容易梯度消失。
  • Softmax——将一组数值转换为概率分布（所有输出之和为 1）。通常用于多分类问题的输出层。
ReLU 函数图像 y
█          █          █
────────────────────────────────→ x
                           0
x < 0 时输出 0，x > 0 时输出 x 本身
为什么 ReLU 这么流行？有两个原因：第一，它的导数非常简单——x > 0 时导数为 1，x < 0 时导数为 0。这意味着在反向传播中梯度不会消失（对于正值区域）。第二，计算成本极低——只需要一个比较操作。在拥有数百层的深度网络中，每一层的计算开销都很重要。

为什么单层感知机不够用？—— XOR 问题

1958 年，Frank Rosenblatt 提出了感知机（Perceptron）——这是第一个真正意义上的"学习算法"。它可以自动调整权重来完成二分类任务。当时的媒体大肆报道，认为感知机会很快让人类实现通用智能。

但 1969 年，Marvin Minsky 和 Seymour Papert 发表了一本名为 Perceptrons 的书，严格证明了单层感知机无法解决 XOR 问题。

XOR 问题是什么？

XOR（异或）是一个简单的布尔运算：两个输入相同输出 0，不同输出 1。

  • 0 XOR 0 = 0
  • 0 XOR 1 = 1
  • 1 XOR 0 = 1
  • 1 XOR 1 = 0

在二维平面上画出来：(0,0) 和 (1,1) 是一类，(0,1) 和 (1,0) 是另一类。你无法用一条直线把它们分开。单层感知机只能做线性分类——它只能画一条直线来分类数据。而 XOR 需要两条线（或者说，需要一个"折线"）。

历史后果：XOR 问题的证明直接导致了连接主义研究的"第一次寒冬"。资助方认为神经网络有根本性缺陷，研究资金大幅减少。直到 1986 年反向传播算法被推广，多层神经网络（MLP）被证明可以解决 XOR 问题，连接主义才重新崛起。这段历史告诉我们：一个理论的局限性不等于整个方向的失败——很多时候需要的只是一个更好的算法。

多层感知机（MLP）——隐藏层的魔力

破解 XOR 问题的方法是：加入隐藏层。一个带有至少一个隐藏层的前馈神经网络可以逼近任何连续函数——这被称为万能近似定理（Universal Approximation Theorem）。

输入层 x₁ x₂ → → 隐藏层（神经元 1） h₁
spacer h₂ h₃ → → 输出层 y

隐藏层做了什么？

  • 特征组合——隐藏层的每个神经元学习输入数据的某种"组合方式"。比如在一个图像识别网络中，第一层隐藏层可能学习到"边缘检测"，第二层学习到"角点检测"，第三层学习到"形状组件"。
  • 非线性变换——每一层都通过激活函数引入非线性，使得网络可以学习越来越复杂的决策边界。
  • 层次化抽象——越靠近输入的层学习越基础的抽象（如像素、边缘），越靠近输出的层学习越高级的抽象（如物体、类别）。

前向传播——数据如何在网络中流动

当一个输入进入神经网络时，它会经过一系列层的计算，最终产生输出。这个过程叫做前向传播（Forward Propagation）。让我们用一个具体的例子来理解：

具体计算示例：一个简单的 2-3-1 网络

网络结构：2 个输入神经元，3 个隐藏神经元，1 个输出神经元。全部使用 ReLU 激活。

输入：x = [0.5, 0.8]

第一步——隐藏层计算：

  • h₁ = ReLU(w₁₁·0.5 + w₁₂·0.8 + b₁) = ReLU(0.3 + 0.4 - 0.2) = ReLU(0.5) = 0.5
  • h₂ = ReLU(w₂₁·0.5 + w₂₂·0.8 + b₂) = ReLU(-0.1 + 0.6 - 0.3) = ReLU(0.2) = 0.2
  • h₃ = ReLU(w₃₁·0.5 + w₃₂·0.8 + b₃) = ReLU(0.2 + 0.1 - 0.4) = ReLU(-0.1) = 0

第二步——输出层计算：

  • y = ReLU(v₁·0.5 + v₂·0.2 + v₃·0 + b_out) = ReLU(0.15 + 0.1 - 0.1) = ReLU(0.15) = 0.15

输出：y = 0.15

关键理解：前向传播是确定性的——给定相同的输入和权重，每次都会得到相同的输出。它不包含"学习"过程。前向传播只是"用当前学到的知识做预测"。学习发生在另一个过程中——反向传播（我们将在下一课详细讲解）。

损失函数——网络如何知道自己"错多远"

前向传播产生输出后，我们需要一个标准来衡量这个输出有多"错"。这就是损失函数（Loss Function）的作用。它把一个预测值和真实值之间的差距量化为一个数字。

两种最常见的损失函数

  • 均方误差（MSE）——适用于回归问题。计算预测值和真实值之差的平方平均值。对大的误差惩罚更重（因为是平方）。
  • 交叉熵损失（Cross-Entropy）——适用于分类问题。衡量预测概率分布与真实标签之间的差异。当预测概率与真实标签差距大时，损失急剧上升。
直觉理解：损失函数就像GPS导航上的"预计到达时间误差"。你预计 30 分钟到达（预测值），实际花了 45 分钟（真实值），损失函数就是告诉你"你偏离了 15 分钟"。神经网络的目标就是通过调整权重来最小化这个"偏离"。

这一课的核心要点

  • 人工神经元模拟生物神经元：接收输入、加权求和、加偏置、通过激活函数输出。
  • 激活函数引入非线性，是神经网络可以学习复杂模式的关键。ReLU 是目前最常用的激活函数。
  • 单层感知机只能做线性分类，无法解决 XOR 问题——这是 1969 年 Minsky 和 Papert 证明的。
  • 加入隐藏层后，神经网络可以学习非线性关系，这被称为万能近似定理。
  • 前向传播是确定性的预测过程；学习发生在反向传播中（下一课）。
  • 损失函数量化预测误差，是优化的目标。
