JTB 理论：被辩护的真信念

知识的传统定义：Justified True Belief

  • 信念（Belief）：相信命题为真
  • 真理（Truth）：命题必须为真
  • 辩护（Justification）：有充分的理由

盖梯尔问题

1963年，Edmund Gettier 证明 JTB 三个条件仍不足以保证知识——可能存在"碰巧为真"的情况。

  • 牛棚案例：看到纸板牛，碰巧真有牛在后面
  • 核心：运气介入 → 不是知识

笛卡尔的恶魔论证

如果恶魔系统欺骗你所有感官，如何确定外部世界存在？

  • 解决方案："我思故我在"——怀疑本身证明主体存在
  • 唯一不可怀疑的命题

知识来源

  • 感知（感官）— 最直观也最不可靠
  • 理性（逻辑推理）— 数学知识
  • 权威（他人证言）— 绝大多数知识的来源
  • 记忆 — 可被篡改
  • 直觉 — 道德和数学中的快速判断

美德认识论

认知美德：开放心态、智力谦逊、思维严谨

  • 主动寻求反面证据
  • 精确自我评估
  • 承认知识边界

社会认识论

知识是社会化建构的

  • 同行评审 = 辩护的社会化形式
  • 判断信源可信度 = 核心认识论技能
  • 集体知识 > 个体知识的总和

日常应用

  • 识别虚假信息 → 追问"怎么知道的？"
  • 对抗确认偏误 → 主动找反面证据
  • 认识论谦逊 → 精确而非傲慢的自我评估
