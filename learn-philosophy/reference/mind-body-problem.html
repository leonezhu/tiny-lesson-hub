快速参考 · 哲学核心概念速查

身心问题 = 物理世界（客观、可测量） vs 主观体验（私密、不可约）

五大立场一览

1. 二元论（Dualism）

  • 心灵和物质是两种不同实体
  • 核心难题：因果互动问题
  • 代表：笛卡尔

2. 同一论（Identity Theory）

  • 心理状态 = 大脑状态
  • 类比：闪电 = 放电
  • 问题：无法解释不同生物体的不同神经实现

3. 功能主义（Functionalism）

  • 心理状态由功能角色定义，不由材料定义
  • 多重可实现性：同一心理状态可在不同基质中实现
  • 支持AI可能有意识的观点
  • 代表：普特南、普莱斯

4. 取消唯物主义（Eliminative Materialism）

  • 日常心理学概念是错误理论
  • 类比：燃素 → 氧化反应
  • 最激进的物理主义版本
  • 代表：丘奇兰德夫妇

5. 属性二元论（Property Dualism）

  • 只有一种实体（物质），但有两种属性（物理+心理）
  • 心理属性不可还原为物理属性
  • 避免了因果互动问题（因为只有一个实体）
  • 代表：查尔莫斯

三大思想实验

中文房间（John Searle, 1980）

一个人在房间里按规则手册处理中文符号，输出看似理解中文的回答。他理解中文吗？

目的：反驳强AI——语法操作 ≠ 语义理解。

玛丽的房间（Frank Jackson, 1982）

全知物理学家玛丽从未见过颜色，第一次看到红色时学到了新东西吗？

目的：论证存在非物理的事实（感质）。

哲学僵尸（David Chalmers, 1996）

物理上完全复制你的人，但没有主观体验。逻辑上可能吗？

目的：如果可能 → 物理主义为假。

关键术语

  • 感质（Qualia） — 主观体验的"是什么感觉"层面
  • 因果封闭性 — 物理世界有完整的物理因果链
  • 多重可实现性 — 同一心理功能可由不同物理基质实现
  • 还原论 — 高级现象可完全解释为低级现象
  • 涌现 — 整体具有部分所不具备的新属性
