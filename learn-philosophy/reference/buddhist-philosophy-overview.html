佛教哲学：缘起、空性与无我

快速参考卡 · 对应课程 第7课 — 佛教哲学

核心框架

  • 四圣谛：苦（问题诊断）、集（原因分析）、灭（解决可能）、道（实践方法）
  • 缘起（pratītyasamutpāda）：一切现象依赖条件而生，没有独立自存的事物
  • 性空（sūnyatā）：事物没有固定不变的自性，但条件性的存在是真实的
  • 无我（anātman）：五蕴（色、受、想、行、识）聚合产生"我"的错觉，无固定实体

关键概念

  • dukkha：不满足、不圆满——不是"人生痛苦"而是"存在层面的不圆满"
  • taṣhā：贪爱——对事物固定不变性的执著，是苦的根源
  • nirvāṇa：涅槃——贪嗔痴三毒的熄灭，不是天堂而是"止息"
  • 八正道：正见、正思维、正语、正业、正命、正精进、正念、正定

主要学派与人物

  • 佛陀（约公元前5世纪）：四圣谛、八正道、五蕴分析
  • 龙树（约150-250年）：中观学派、四句否定、二谛论
  • 世亲（约4-5世纪）：唯识学派、八识理论
  • 智顗（538-597年）：天台宗、一念三千
  • 一行禅师（1926-2022年）：互即互入（interbeing）、生态哲学

现代连接

  • 心理学：正念冥想、CBT、MBSR 直接源于佛教观照实践
  • 认知科学：叙事自我（丹尼特）与佛教无我理论的共鸣
  • 神经科学：长期禅修改变大脑前额叶皮层和岛叶结构
  • AI研究："执念"与过拟合的哲学类比
