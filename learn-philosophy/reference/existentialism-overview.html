存在主义 · 参考卡

哲学 · 快速参考

核心命题：存在先于本质 — 人没有预设的本质，通过选择和行动定义自己。

四大核心概念

自由与责任

人被判定为自由。每一刻都在选择——包括"不选择"。不能把责任推给外部因素。

焦虑（Angst）

焦虑是自由的眩晕，不是病。逃避焦虑等于逃避自由。学会带着焦虑行动。

荒诞（Absurd）

人渴望意义，世界沉默以对。荒诞是关系而非属性。加缪的答案：反抗——明知无意义，仍然投入生活。

本真性（Authenticity）

清醒认识自己的自由与选择，不活在"大家都这样"的随大流中。海德格尔：面对死亡才能最本真地活着。

三位关键人物

克尔凯郭尔

存在主义之父。"真理是主观的"——需要你用整个生命去投入的真理。焦虑是自由的信号。

萨特

最系统化的存在主义。存在先于本质 → 绝对自由 → 绝对责任 → 焦虑 → 本真性。"自欺"是最大敌人。

加缪

荒诞中的反抗者。西西弗斯推石头——明知会失败，仍然推。"必须想象西西弗斯是幸福的。"

关键概念速查

  • 自欺（mauvaise foi）：用虚假理由逃避自由，如"我只能这样"
  • 他者即地狱：他人通过凝视把你变成客体，你失去自由
  • 信仰的跳跃：从理性推理走向信仰决定，没有中间路线
  • 荒诞：人与世界冲突的关系状态，非世界本身的属性

与斯多葛对比

  • 斯多葛：宇宙有理性秩序 → 顺应自然 → 追求平静
  • 存在主义：宇宙沉默无意义 → 创造意义 → 拥抱焦虑
  • 共同点：个人责任、不怪外部、关注"如何活"

推荐阅读

  • 《存在与虚无》— 萨特（系统阐述）
  • 《西西弗斯神话》— 加缪（荒诞与反抗）
  • 《第二性》— 波伏娃（存在主义+女性主义）
  • 《恐惧与战栗》— 克尔凯郭尔（信仰与焦虑）
  • SEP — Existentialism（学术综述）
