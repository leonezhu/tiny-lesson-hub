四种接触层次

1. 借词（Lexical Borrowing）

直接借用词汇，最常见、最表层。

  • pizza ← 意大利语, tsunami/emoji ← 日语, schadenfreude ← 德语
  • 语音本土化：外来词按母语音系规则调整（coffee → コーヒー）

2. 语音借用（Phonological Borrowing）

改变母语音位系统。

  • 韩语/日语借入送气清音 /pʰ/
  • 英语 /θ/ /ð/ 在其他语言中被替换为 t/s

3. 结构借用（Structural Borrowing）

借用语法或句法结构，深层接触。

  • 爱尔兰语从 VSO 变为 SVO 语序（受英语影响）

4. 深度接触

  • 皮钦语 Pidgin → 语法极简的临时工具
  • 克里奥尔语 Creole → 皮钦语成为母语后扩展为完整语言
  • 代码切换 Code-switching → 同一对话中切换多种语言

英语三层词汇

日耳曼 day, house, eat, love, water — 基础生活词汇
法语 justice, government, beef, pork, music — 法律/军事/烹饪/艺术
拉丁/希腊 chemistry, hypothesis, entropy, system — 学术/科学/医学

汉语三次借词潮

1. 佛经翻译（1-6世纪）：菩萨、世界、刹那、因果 ← 梵语

2. 明清之际：几何、代数、逻辑 ← 欧洲语言

3. 近代回流：社会、经济、科学、哲学 ← 日语借词回流

关键概念

  • 语言纯净性 = 神话。所有自然语言都在不断借用。
  • 接触方向 ≈ 权力结构。强势语言输出词汇，弱势语言输入。
  • 借词传播 ≈ 技术扩散曲线（从早期采用者扩散）。
  • 水平基因转移 ≈ 语言借用（跨语言交换，非垂直传承）。
