句法学

第 4 课 · ~12 分钟 · 语言学的核心模块

学完本课之后，你将能够：识别句子的基本结构成分（主语、谓语、宾语等），理解不同语言的语序差异（如 SVO、SOV、VSO），掌握生成语法的基本思想，并能分析简单和复杂句子的结构层次。

什么是句法学？

句法学（Syntax）研究的是词语如何组合成合格句子的规则系统。如果说语音学研究声音，形态学研究单词内部结构，那么句法学研究的就是单词之间的关系——谁修饰谁，谁是主干，什么排列顺序是可接受的。

一个核心直觉：人类语言有无限多的句子，但规则是有限的。你从未听过"张三昨天在图书馆因为要准备考试所以读了那本关于量子力学的书"这句话，但你完全能理解它——因为你脑子里有一套隐性的句法规则。

句法的核心问题

句法学试图回答三个基本问题：

  1. 合格性：什么样的句子是合格的？什么是不合格的？
  2. 结构：句子的内部结构是什么？哪些词组成短语，哪些短语组成更大的单位？
  3. 普遍性：所有人类语言共享哪些句法规则？哪些是可以变化的？

核心概念一：句法成分与短语结构

每个句子都由更小的单位层层嵌套构成。最基本的短语类型包括：

短语类型 缩写 构成方式 例子
名词短语 NP 限定词 + 形容词 + 名词 那个穿红衣服的学生
动词短语 VP 动词 + 宾语/补语 读了那本书
形容词短语 AP 程度副词 + 形容词 非常漂亮
介词短语 PP 介词 + 名词短语 在桌子上
句子 S / IP / TP 主语 + 谓语 他来了
📌 结构拆解示例
"那个学生认真地读了一本书"
NP（那个学生）+ AP（认真地）+ VP（读了一本书）
VP 内部：V（读）+ NP（一本书）

核心概念二：语序类型

世界上的语言在基本语序上差异巨大。句法学的一个重大发现是：虽然表面看起来千差万别，但所有语言的语序都可以归入有限的几种类型。

语序类型 代表语言 示例
SVO（主-动-宾） 汉语、英语、法语、西班牙语 我（S）吃（V）苹果（O）
SOV（主-宾-动） 日语、韩语、土耳其语、藏语 我（S）苹果（O）吃（V）
VSO（动-主-宾） 古典阿拉伯语、爱尔兰盖尔语 吃（V）我（S）苹果（O）
VOS（动-宾-主） 马达加斯加语、斐济语 吃（V）苹果（O）我（S）
OSV（宾-主-动） 少数南美原住民语言 苹果（O）我（S）吃（V）
重要发现：汉语和英语都是 SVO 语序，但汉语比英语灵活得多。汉语可以说"苹果我吃了"（话题化），而英语不能说 *"Apples I ate"（除非在特定语境）。这说明同一语序类型内部，各语言的句法规则仍有显著差异。

核心概念三：生成语法与乔姆斯基

1957 年，诺姆·乔姆斯基出版了《句法结构》，彻底改变了语言学。他的核心观点是：人类天生拥有一套"普遍语法"（Universal Grammar, UG）——一种内在于大脑的语言机制，使得儿童能够在有限输入的情况下快速学会任何人类语言。

生成语法的核心思想

深层结构 vs 表层结构：同一个意思可以通过不同的句子表达。例如：

"张三打了李四" ↔ "李四被张三打了"
深层结构相同（某人打某人），表层结构不同（主动 vs 被动）

转换规则：从深层结构到表层结构需要经过一系列转换操作，如移位、删除、合并等。

管约论（GB）→ 最简方案（MP）：乔姆斯基的理论经历了多次迭代，核心始终不变——寻找最少的规则解释最多的现象。

核心概念四：依存关系与题元角色

除了短语结构，句法学还关注词语之间的语义-句法关系。动词决定了它的论元（arguments）需要什么类型的参与者。

题元角色（Thematic Roles）
"妈妈给了孩子一个苹果"
妈妈 = 施事（Agent，动作发出者）
孩子 = 与事（Recipient，接受者）
苹果 = 受事（Theme，被移动的对象）

不同的动词要求不同的题元角色组合。比如"睡"只需要一个参与者（"我睡了"），而"给"需要三个参与者（"我给某人某物"）。如果一个句子缺少必需的论元，它就不合格：

合格 vs 不合格
✓ 我睡了。（"睡"只需要一个论元）
✗ 我给。（"给"需要至少两个论元，不合格）
✗ 我给了。（同上）
✓ 我给了他钱。（完整，合格）

核心概念五：从句与嵌套

句法学最迷人的地方在于递归性（recursion）——你可以把一个句子嵌入另一个句子，理论上可以无限嵌套下去：

递归嵌套示例
我知道你知道他知道我喜欢这个。
我知道 [ 你知道 [ 他知道 [ 我喜欢这个 ] ] ]
那个 [ 昨天 [ 在 [ 图书馆 ] 里 ] 读 [ 那本 [ 关于 [ 量子力学 ] 的书 ] ] ] 的人来了。
多层定语从句/修饰语的嵌套
为什么这很重要？递归性是语言区别于大多数动物通信系统的核心特征之一。蜜蜂的舞蹈、猴子的警报叫声都无法像人类语言一样无限嵌套。乔姆斯基认为这是人类语言独有的认知能力。

常见错误与注意事项

中国学习者常见的句法误区

1. 英语中的"主题突出" vs "主语突出"

汉语：那本书，我看完了。（话题优先）
英语不能说 *"That book, I finished."（口语中偶见，但不合标准语法）
英语必须说：I finished that book. 或 That book has been finished by me.（被动）

2. 英语的从句必须有引导词

✗ 我不知道他去不去。（直译 I don't know he go or not）
✓ I don't know whether he is coming or not.
英语的间接疑问句必须用 whether/if 引导，不能省略

3. 英语的关系从句位置

汉语：我昨天遇见的那个学生
英语：the student whom I met yesterday
汉语的定语在名词前，英语的关系从句在名词后

句法学在现实中的应用

理解句法不只是学术训练，它在多个领域都有直接价值：

应用场景

  1. NLP / AI：解析器（parser）依赖句法规则将文本转化为结构化表示，是机器翻译、问答系统的基础。
  2. 二语教学：知道目标语的句法类型（如英语是 SVO、关系从句后置），可以预测中国学习者的难点。
  3. 法律与合同：长难句的句法分析帮助精确理解条款含义，避免歧义。
  4. 语言学调查：描述一种无文字语言的句法结构，是记录和保存语言多样性的关键。
  5. 认知科学：失语症患者的句法损伤模式（如 Broca 区损伤影响句法加工但不影响语义），揭示了大脑中语言处理的模块化组织。

练习：检验你的理解

以下 8 道题涵盖本课核心概念。每题选择一个答案，点击后查看解释。

Q1
考察语序类型识别
日语"私はりんごを食べる"（watashi wa ringo o taberu）属于哪种基本语序？
Q2
考察短语结构识别
在短语"非常漂亮的红色雨伞"中，"非常"修饰的是哪个成分？
Q3
考察合格性判断
下列哪个句子在标准汉语中是合格的？
Q4
考察深层结构与表层结构
"猫追老鼠"和"老鼠被猫追"在生成语法中有什么关系？
Q5
考察语言普遍性
关于人类语言语序，以下哪项是正确的？
Q6
考察递归性理解
"我知道你知道他知道我喜欢这个"这句话体现了句法的什么特性？
Q7
考察题元角色
在句子"老师寄给学生一封信"中，"学生"的题元角色是？
Q8
考察中英句法差异
英语说 "the man who I saw"，而汉语说 "我看见的那个男人"。这反映了什么差异？
