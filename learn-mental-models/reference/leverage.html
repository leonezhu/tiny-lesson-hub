杠杆

参考卡 · 多元思维模型 · Lesson 12

核心公式

产出 = 努力 × 杠杆率

杠杆不是"走捷径"，而是找到正确的系统位置让小力量产生大结果。

四种杠杆类型

1. 劳动力杠杆

让别人为你工作。最原始、最难管理的杠杆。管理 N 个人的难度远超 N 倍。

2. 资本杠杆

用钱生钱。通用性强但需要"借"——有成本。过度杠杆化会导致无限下行风险。

3. 代码杠杆

写一次，运行无限次。边际成本为零，具备"睡眠能力"。21 世纪最强大的杠杆。

4. 媒体杠杆

内容一次创作，影响无限人。边际成本为零但需要"观众"变量。分发竞争激烈。

关键原则

  • 先用零杠杆把事做对，再用杠杆放大
  • 在低杠杆率的事上高杠杆 = 更快地做错事
  • 代码和媒体不需要别人同意就能使用——最佳的个人杠杆
  • 杠杆放大一切：好判断和好能力被放大，坏判断和坏能力也被放大
  • 最安全的杠杆是那些不需要"还"的：代码和媒体

与其他模型的连接

  • 复利：杠杆在金融领域的具体应用；代码/媒体也有累积效应
  • 临界质量：媒体/资本需要跨越阈值才能产生指数级影响
  • 第一性原理：选择杠杆前必须先理解真正的价值创造是什么
