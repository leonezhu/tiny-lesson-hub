经典条件反射 × 操作性条件反射 · 第 7 课

核心定义

行为主义主张心理学只研究可观察的行为，认为行为是环境刺激与反应之间的联结。通过控制刺激可以预测和控制行为。

经典条件反射（巴甫洛夫）

Classical Conditioning — 通过关联两个刺激形成自动反应

  • 食物 → 流口水（天然）
  • 铃声 + 食物反复配对 → 铃声 → 流口水（学会）
  • 解释：恐惧症、味觉偏好、广告中的情绪关联

操作性条件反射（斯金纳）

Operant Conditioning — 行为的结果决定该行为未来是否重复

  • 正强化： 给奖励 → 行为增加
  • 负强化： 移除厌恶刺激 → 行为增加
  • 惩罚： 给厌恶刺激 → 行为减少
  • 消退： 不再强化 → 行为减少

强化时间表

  • 连续强化： 每次都给 → 学习快，消退也快
  • 间歇强化（可变比率）： 随机给 → 学习慢，行为极其持久
  • 应用：老虎机、抽卡游戏、社交媒体推送、无限滚动

关键人物

  • 巴甫洛夫： 经典条件反射，狗唾液实验
  • 华生： 行为主义宣言，"给我一打婴儿"
  • 斯金纳： 操作性条件反射，斯金纳箱，强化时间表
  • 米尔格兰姆： 服从实验，权威与道德

现实应用

  • 教育：形成性评价、及时反馈
  • AI：强化学习（AlphaGo、自动驾驶）
  • 产品设计：间歇强化维持用户粘性
  • 行为矫正：治疗恐惧症和成瘾

局限性

  • 忽略认知过程（思想、信念、预期）
  • 忽视内在动机（自我决定论的批评）
  • 实验室结论不一定适用于复杂社会情境
