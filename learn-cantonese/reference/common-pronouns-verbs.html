常用代詞與動詞速查卡

第3課 · 粵語核心詞彙速查

代詞

粵語粵拼普通話
我ngo5我
你nei5你
佢keoi5他/她/它（不分性別）
我哋ngo5 dei6我們
你哋nei5 dei6你們
佢哋keoi5 dei6他們
邊個bin1 go3誰
呢個ni1 go3這個
嗰個go2 go3那個
呢度ni1 dou6這裡
嗰度go2 dou6那裡
哋 = 複數標記（們）；啲 = 複數標記（些）+「一些」

核心動詞

粵語粵拼普通話例句
食sik6吃食咗飯未？
飲jam2喝飲杯茶。
行haang4走行路上班。
睇tai2看睇電視／睇新聞
聽ting1聽聽歌
講gong2講講嘢
做zou6做做乜嘢？
去heoi3去去邊度？
嚟lai4來快啲嚟！
有jau5有有冇睇？
係hai6是佢係我同學。
鍾意zung3 ji3喜歡好鍾意食粵菜。
識sik1認識/會識講廣東話
想soeng2想想返屋企。
得dak1可以得唔得？
否定形式：唔 = 不（否定習慣/狀態）；冇 = 沒有（否定完成）；唔好 = 不要

完成體標記

標記粵拼功能例句
咗zo2動作已完成食咗飯 = 吃了飯
未mei6句尾：還沒有食未？= 吃過了嗎？
