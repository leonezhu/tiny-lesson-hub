社交寒暄与敬语

第九课 · ~12 minutes · 日常会话基础

本节课目标：学会粤语中最常用的社交寒暄用语和敬语表达，包括见面问候、告别、感谢致歉、称呼长辈和职场敬语——让你在真实社交场合说得自然、听得明白。

为什么这课很重要

很多学粤语的人能点餐、能讲价，但跟长辈打招呼时会紧张，在职场中不知道该用哪种语气。粤语的社交用语和普通话差别很大——不仅仅是词汇替换，还包括称呼方式、语气助词和敬语层级。这些看似"不重要"的寒暄，实际上决定了别人觉得你有冇礼貌（有没有礼貌）。

见面问候

👋 基础问候

场景粤语普通话说明
通用问候你好吗？
nei5 hou2 ma3?		你好吗？字面相同，但语调不同。粤语的"好"声调更高。
最近怎样最近点呀？
zui3 gan2 dim2 aa3?		最近怎么样？点 = 怎样，呀 = 语气词，软化语气。
很久不见好久冇见呀！
haau2 ma5 mou5 gin3 aa3!		好久不见！冇 = 没有，比普通话更口语化。
食咗饭未？食咗饭未？
sik6 zo2 faan6 mui6?		吃了吗？粤语最经典的寒暄语。不是真问你吃没吃，而是"你好"的粤语版。
文化提示："食咗饭未？"是粤语区最高频的寒暄语，地位等同普通话的"吃了吗"。但不同之处是：回答时可以很简短（食咗喇 / 吃了）、也可以反向问候。这反映了岭南饮食文化中"吃"的核心地位。

告别用语

👋 道别方式

场景粤语普通话说明
通用告别拜拜！
baak3 baak3!		拜拜！来自英语 bye-bye，最通用。
先走了我先走啦！
ng5 sin1 zou2 laa3!		我先走了！啦 = 了+啊，让语气更柔和。
回头见回头见！
wai4 tau4 gin3!		回头见！朋友间随意使用。
保重保重啊！
bou2 zung3 aa3!		保重！较正式的告别，用于长期不见。
慢走慢走啊！
maan6 zau2 aa3!		慢走！送别对方时用的客套话。

感谢与致歉

🙏 感谢表达

程度粤语普通话说明
一般感谢多谢！
do1 ze6!		谢谢！最常用。
非常感谢多谢晒！
do1 ze6 saai3!		非常感谢！晒 = 全部，表"彻底感谢"。
不客气冇问题！
mou4 man6 tai4!		不客气！也可说：应该嘅（ying1 goi1 ge3）
劳驾了劳烦你！
lou4 faan4 nei5!		劳驾！请求帮忙时使用。

😔 道歉表达

程度粤语普通话说明
小歉意唔好意思！
m4 hou2 ji3 si6!		不好意思！最通用的道歉。
正式道歉对唔住！
deoi3 m4 zyu6!		对不起！更正式，用于真正的错误。
真的抱歉真係好抱歉！
zhen1 hai6 hou2 pu3 jiu6!		真的很抱歉！强调句，真係 = 真的。
关键区别：普通话的"不好意思"和"对不起"在粤语中是分得很清楚的两个词。唔好意思用于轻微的冒犯（比如踩到别人的脚、打断别人说话），对唔住用于真正的过失（比如迟到很久、弄坏了别人的东西）。用错了会显得不够真诚或过于隆重。

称呼系统

👤 人称与称呼

粤语的称呼系统是比普通话更精细的社交工具，尤其是对长辈、上级和不太熟的人：

关系粤语称呼普通话使用场景
年长男性阿叔 / 伯伯叔叔 / 伯伯称呼父亲的兄弟或邻居长辈
年长女性阿姨 / 姐阿姨 / 姐姐（方姐）用于称任何年长女性，非常常用
年轻人細佬 / 細妹弟弟 / 妹妹也可用于称呼不熟的年轻人
同事同事同事粤语中直接称呼对方为"XX同事"很自然
老师/师傅师傅 / 老师师傅 / 老师对专业人士（医生、律师、教练）用"师傅"更亲切
自己我我粤语中很少用谦称，直接用"我"即可

🤝 职场敬语

粤语职场中有几组高频敬语，掌握它们能让你的专业形象大大加分：

场景粤语普通话
请多指教请多关照！
cing2 do1 gou1 ziu3!		请多关照！
辛苦了辛苦晒！
san1 hou2 saai3!		辛苦了！
不好意思打扰了打扰晒啦！
daau2 jau2 saai3 laa3!		打扰了！
麻烦你了麻烦你啦！
ma6 faan3 nei5 laa3!		麻烦你了！
请问请问一下
cing3 maan5 jat1 ha6		请问一下
搞掂就通知你搞掂就同我讲！
gaau2 dim6 zau6 tung4 ng5 gong2!		完成就告诉我！
语气助词的魔力：注意上面的表达中频繁出现的晒和啦。晒表示"全部"，用在感谢或辛苦后面（多谢晒、辛苦晒），让语气更真诚更强烈。啦是"了+啊"的组合，让句子从"冷冰冰陈述"变成"带温度的交流"。省略语气助词，粤语听起来就像在读说明书。

完整对话示例

🏢 职场晨间寒暄

A：朝早！最近点呀？

B：朝早！还好呀，你呢？

A：都係啦，最近忙紧个新项目。

B：加油啊！有咩需要帮忙嘅同我讲啦。

A：多谢晒！你人真好。


翻译：

A：早上好！最近怎么样？

B：早上好！还好啦，你呢？

A：都一样，最近在忙新项目。

B：加油！有什么需要帮忙的跟我说。

A：非常感谢！你真好。
